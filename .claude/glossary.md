>
> 原則として **ゲーム内 UI 表示で使われている公式日本語ローカライズに準拠**。Steam JP ストア・Pocketpair 公式ページ・実機 UI で確認できないものは「【要検証】」フラグを付けています。
>
> 表記方針: 重要な固有名詞は **「日本語（English）」のバイリンガル併記** を初出時に行い、文脈に応じて簡略化します。

## 表記運用ルール

1. **初出時はバイリンガル併記**：「サーベル（Saber）」「レイピア（Rapier）」のように、最初の登場時は日本語と英語両方を示す
2. **2回目以降は文脈で判断**：表内なら英語のみで簡潔に、文章なら日本語で読みやすく
3. **【要検証】フラグの扱い**：実機 UI 表示が確認でき次第、フラグを外して出典を「実機 UI」に更新する
4. **ユニーク武器・キャラクター名**：原語維持を基本とし、必要に応じて意訳を併記
5. **新規記事執筆時は本ページの訳語を参照**
