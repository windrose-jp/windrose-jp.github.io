# 鉱石・インゴット

> 情報源: [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/)

## 鉱石・インゴット一覧

### 銅（Copper）
序盤に最初に入手する金属素材です。

| 項目 | 内容 |
|------|------|
| 採掘 | 銅鉱脈をつるはしで採掘 |
| 精錬 | 製錬炉（Smelting Furnace）で銅インゴット（Copper Ingot）に精錬 |
| 用途 | 武器・道具クラフト、建築 |
| 備考 | 銅採掘後にDoctorがボートを提供してくれる |

---

### タンバガ（Tumbaga）
中級金属素材。銅より上位の素材です。

| 項目 | 内容 |
|------|------|
| 入手方法 | 情報収集中 |
| 精錬 | 製錬炉（Smelting Furnace）でタンバガインゴット（Tumbaga Ingot）に精錬 |
| 用途 | 武器鍛冶台（Weaponsmith Station）での武器アップグレード |
| 備考 | 第一島で見つかる場所が確認されている |

---

その他の鉱石種類（鉄・金など）・入手場所・採掘条件は情報収集中です。
