# 鉱石・インゴット

> 情報源: [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/) / [TechRaptor Resources Guide](https://techraptor.net/gaming/guides/windrose-resources) / [GPORTAL Beginner's Quickstart](https://www.g-portal.com/wiki/en/windrose-beginners-quickstart-guide/)

## 採掘の基本

鉱石の採掘には **つるはし（Pickaxe）** が必要です。ツールは以下の階層で強化されます:

1. **Stone Pickaxe（石のつるはし）** — 序盤、Copperまで対応
2. **Copper Pickaxe（銅のつるはし）** — Iron対応
3. **Iron Pickaxe（鉄のつるはし）** — 上位鉱石対応

鉱石は製錬炉（Smelting Furnace）で **インゴット** に加工してから使用します。精錬にはCharcoal Kilnで作った木炭が燃料として必要です。

> ⚠️ **重要**: **焚き火（Bonfire）の半径内では鉱石がリスポーンしない**。採掘拠点と生活拠点は分離して運用すること。詳細は [リスポーン仕様](../exploration/respawn.md) を参照。

## 銅（Copper）

序盤に最初に入手する金属素材。

| 項目 | 内容 |
|------|------|
| 採掘場所 | **Coastal Jungleの銅鉱脈** / 洞窟内 |
| 必要ツール | Stone Pickaxe以上 |
| 精錬 | Smelting Furnace で Copper Ingot に精錬 |
| 用途 | 武器・道具クラフト、Copper Axe/Pickaxeへのアップグレード、建築 |
| 備考 | 銅採掘後にDoctorがボートを提供してくれる |

**戦略**: 序盤の主要目標。銅を入手すると道具ティアが上がり、Iron鉱石の採掘が可能になる。また **ランタン（Lantern）** のクラフトも解放される。

---

## 鉄（Iron）

中盤の主要金属素材。

| 項目 | 内容 |
|------|------|
| 採掘場所 | **Foothills Mine 専用**（Coastal Jungleでは入手不可） |
| 必要ツール | Copper Pickaxe以上 |
| 精錬 | Smelting Furnace で Iron Ingot に精錬 |
| 用途 | 上位装備・船の釘（Iron Nail）・弾丸（Iron Bullet） |

**重要**: **Coastal Jungleでは鉄素材が入手できない**。鉄ティアの装備が欲しくなったら Foothills のMineに向かう必要がある。

---

## タンバガ（Tumbaga）

中級金属素材。銅の上位互換的な素材。

| 項目 | 内容 |
|------|------|
| 入手方法 | 特定の鉱脈から採掘（詳細は情報収集中） |
| 精錬 | Smelting Furnace で Tumbaga Ingot に精錬 |
| 用途 | **Weaponsmith Stationでの武器アップグレードの主要素材** |
| 備考 | 第一島でも見つかる場所が確認されている |

---

## Ancient系（古代素材）

高レアリティの古代遺跡系素材。Cursed Swamps等の高難度エリアで入手。

| 素材名 | 入手場所 | 用途 |
|-------|--------|------|
| Ancient Scraps（古代のスクラップ） | 沼の遺跡 | 上位装備素材 |
| Ancient Steel Shard（古代の鋼片） | 遺跡 | 上位武器素材 |
| Ancient Relic Fragment（古代遺物の断片） | 遺跡・ダンジョン | 特殊クラフト |
| Aquamarine（アクアマリン） | **Ancient Vault（古代の地下室）** | 希少素材 |

## 精錬の流れ

1. **鉱石を採掘** — Pickaxeで鉱脈を叩く
2. **Charcoal Kilnで木炭を製造** — Wood → Charcoal
3. **Smelting Furnaceに鉱石と木炭を投入** — 時間経過でインゴット完成
4. **インゴットを使ってクラフト** — 武器鍛冶台やWeaponsmith Stationで装備作成

---

## 推奨進行順

1. Stone Pickaxe → Copper Ore採掘（Coastal Jungle）
2. Copper Ingot → Copper Pickaxe製作
3. Foothillsに進出 → Iron Ore採掘
4. Iron Ingot → 上位武器・船装備
5. Cursed Swampsで Ancient系素材を収集
