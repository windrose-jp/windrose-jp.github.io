## 素材収集の基本

![商人・資源交換](../images/merchant-resources.jpg)

- 木材と石材は拠点建設の基本素材
- 鉱石は製錬炉（Smelting Furnace）でインゴットに加工
- 植物・食材は料理・錬金術に使用
- ヤシの木は食材源として保護推奨（切り倒さない）
