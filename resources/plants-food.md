# 植物・食材

> 情報源: [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/)

## 植物・食材一覧

### 癒しのハーブ（Healing Herb）
回復薬の素材として重要な植物です。

| 項目 | 内容 |
|------|------|
| 入手場所 | 島の草地・森など（採取） |
| 主な用途 | ポーション・回復薬のクラフト素材 |
| 備考 | 第二島にまとまった採取スポットが存在（15個前後） |

---

### ヤシの実（Coconut）
ヤシの木から採取できる食材です。

| 項目 | 内容 |
|------|------|
| 入手方法 | ヤシの木から採取（木を切り倒さずに実のみ採る） |
| 主な用途 | ヤシミルク・料理の素材 |
| 注意 | ヤシの木は切り倒さないこと。食材源として継続的に利用するため |

---

### ミスティオーキッド（Misty Orchid）
霧の蘭。錬金テーブルの解放に必要な特別な植物です。

| 項目 | 内容 |
|------|------|
| 入手場所 | 第二島 |
| 用途 | 錬金テーブル（Alchemy Table）の解放 |

---

その他の食材・植物（果物・キノコ類など）・バフ効果・栽培システムは情報収集中です。
