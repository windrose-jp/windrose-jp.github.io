# ポイズ・スタミナ

> 情報源: [Slashskill Hidden Mechanics](https://www.slashskill.com/windrose-tips-and-tricks-hidden-mechanics-the-game-never-explains/) / [Steam: How to Git Good](https://steamcommunity.com/app/3041230/discussions/0/757304900494422048/) / [Neonlightsmedia Stamina Guide](https://www.neonlightsmedia.com/blog/windrose-increase-stamina-guide)

## スタミナ（Stamina）

Windrose の**最重要リソース**。攻撃・回避・ダッシュ・ジャンプ・水泳の全てに消費される。

### 3段階の状態

| 状態 | 説明 | 対処 |
|------|------|------|
| 青（通常） | 安全。通常通り行動できる | 維持するよう立ち回る |
| 黄（注意） | 残量低下 | 攻撃を控えて回復を待つ |
| **赤点滅（Winded）** | **枯渇。大ペナルティ発生** | **即離脱して完全回復** |

### ⚠️ Winded（枯渇）状態のペナルティ

赤点滅状態に入ると以下のペナルティ：
- **スプリント不可**
- **ドッジ不可**
- **スタミナ回復が通常より遅くなる**

この状態で攻撃を受けると大抵一撃で致命的。**20〜30%残した段階で攻撃を止める**のが安全運用。

### スタミナ消費を抑える装備・タレント

| 手段 | 効果 |
|------|------|
| **Flibustier's Attire (2P)** | 攻撃スタミナコスト -20% |
| **Marksman's Rig (2P)** | Sprint/Jump/Dash スタミナ -30% |
| **Agile** タレント | ドッジ・ジャンプのスタミナ削減 |
| **Marathon Runner** タレント | ダッシュ・ドッジのスタミナ削減（全ビルド最優先級） |

### Rested バフ（最重要）

Bonfire 付近で取得する**スタミナ回復速度と最大量を大幅に上げる**バフ。詳細は [食事・ポーション](../character/buffs.md#restedバフ) を参照。

- 裸の Bonfire だと数分程度
- デコレーション配置で **最大30分** まで延長
- 遠征前に必ず取得してから出発

### 重量ペナルティも要注意

持ち物の重量割合によってスタミナ消費・回復が変わる（詳細は [ステータス](../character/stats.md#重量encumbranceペナルティ)）：

- **75%超**: スタミナ回復 -20%
- **100%超**: ドッジのスタミナ消費 **2倍**

## ポイズ（Poise / Guard Shield）

**怯み耐性を表すシールド**。プレイヤーも敵も持っている。画面下のタン色シールドアイコンで確認可能。

### シールドの仕組み

- プレイヤーも敵も **2〜3枚のシールド** を保持
- **シールド0でスタガー**（人間はよろけ、動物は転倒）
- **スタガー中は通常の何倍ものダメージが通る** ← 大ダメージのチャンス

### シールドを削る手段

| 手段 | 削り効率 |
|------|---------|
| **パリィ成功** | **1シールド即削り**（最強） |
| **Heavy Attack** | 大（ライトの約1.7倍） |
| Light Attack | 小 |
| 特定のダッシュアクション | 中 |

**パリィが最も効率よく敵のポイズを削る**。しかもパリィは自分のスタミナを消費しない（成功時）。

### ブロックの仕様

- **シールドは消費するが、スタミナは消費しない**
- シールドが残る限り連続ブロック可能
- **前方のみ防御** → 背後から囲まれると無力
- 赤いエフェクト（Unblockable）は**ブロック・パリィともに不可**

### 戦術的活用

1. **最初にパリィ or Heavy でシールドを削り切る**
2. **シールド破壊後のスタガー中に集中攻撃**（ここが最大ダメージタイミング）
3. **自分のシールドが削られないよう被弾を最小限に**
4. ボス戦は特にシールド管理が勝敗を決める
