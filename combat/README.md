# 戦闘概要

> 情報源: [Steam ストアページ](https://store.steampowered.com/app/3041230/Windrose/) / [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/)

Windroseの戦闘は**ソウルライク（Souls-lite）スタイル**を採用しています。敵の攻撃パターンを読み、パリィ・回避・スタミナ管理を駆使して戦うことが求められます。

## 戦闘の基本コンセプト

- **スタミナが全て**: 攻撃・回避・ダッシュ全てにスタミナを消費。枯渇させると行動不能になりやすい
- **ポイズシステム**: 敵のポイズ（怯み耐性）を削ることで転倒させて大ダメージ
- **リスク管理**: 強引に攻めず、安全なヒット＆ランを基本とする
