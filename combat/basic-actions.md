## 基本操作早見表

| アクション | キー |
|----------|-----|
| **ライト攻撃** | 左クリック |
| **ヘビー攻撃** | **中クリック（マウスホイール押し込み）** ← 見落としがち |
| ブロック | 右クリック |
| パリィ | 右クリック（タイミングよく） |
| **ドッジ（回避）** | **CTRL** |
| ジャンプ | SPACE |
| スプリント | SHIFT |
| 特殊攻撃 | F |
| リロード | R |
| ターゲットロック | T |
| ターゲット切替 | [ または ] |

### ⚠️ ヘビー攻撃は中クリック

多くのプレイヤーが見落とす操作。**マウスホイール押し込み**で発動。ゲーム内チュートリアルで明示されない。

## ライト攻撃 vs ヘビー攻撃

| 項目 | Light | Heavy |
|------|-------|-------|
| ダメージ | ベース | **ライトの約1.70倍（+69.52%）** |
| 体幹削り | 少 | **大（素早く敵シールドを割る）** |
| スタミナ消費 | 低 | やや高い（倍ではない） |
| モーション | 速い | 遅い |
| **キャンセル可否** | **不可** | **不可** |

- **攻撃アニメーションはキャンセル不可**: スパム押しすると動けなくなる罠
- 基本コンボ: **Light 1〜2発 → 離脱 → 隙を見て Heavy**
- 敵の体幹（Guard Shield）を削るなら Heavy が圧倒的に効率的

## パリィ（Perfect Block）

- **成功でスタミナ消費ゼロ**（最大の強み）
- 成功後は **リポスト（反撃）ウィンドウ約2秒**
- 敵のシールドを**即1個削る**
- タイミング: 敵の振り下ろし直前（目安 0.3秒前）
- 早すぎ → 通常ブロック扱い / 遅すぎ → フルダメージ

### パリィ性能は武器で全然違う

| 武器 | パリィ性能 |
|------|----------|
| **サーベル（Saber、特にSturdy Saber）** | **最適**。シールド消費なし、+1ポスチャーボーナス |
| **Razor（レイピアの Ascend）** | **ほぼ連続で成功**可能。Ascendでウィンドウ拡大 |
| レイピア（Rapier、通常） | 可能だがシールド半分消費 |
| グレートソード（Greatsword） | 高Stabilityで chip damage 減少 |
| **クラブ（Club） / ハルバード（Halberd） / Crude系** | **パリィ不可 or 最低性能** |

筋力（Strength）系（重武器）は基本的にパリィに向かない。パリィ運用したいなら **サーベル / Razor** を主軸にする。

## 回避（ドッジ）

- **CTRL** で発動
- i-フレーム（無敵時間）の具体数値は**未公開**
- スタミナ消費大
- **100% 超の重量時はスタミナ消費2倍**
- **赤いエフェクトの攻撃（Unblockable）はドッジ必須** ← パリィ・ブロック不可

### ⚠️ 赤いエフェクト攻撃 = Unblockable

ソウルライク共通のシグナル。**赤く光る攻撃は絶対にブロック・パリィできない**。ドッジで避けるしかない。知らないとボス戦で即死する。

## ブロック

- 右クリックで構え
- **シールド（体幹）を消費するがスタミナは消費しない** → シールド残る限り連続ブロック可
- **前方のみ防御**。背後からの攻撃には効かない
- シールド0になるとスタガー → 無防備

## Rally（反撃回復 / Temporal Health）

ソウルライク定番の「**被ダメ直後に反撃でHPを取り戻す**」機構。

- 被弾直後に**短いウィンドウ内で攻撃を当てるとHPの一部が戻る**
- リソース消費ゼロ = **ゲーム内で最強のリアルタイム回復**
- タレント **You Will Answer for This** で回復量増加
- ウィンドウの秒数・回復%は非公開

回復優先度（[初心者Tips](../tips.md) 参照）:
1. **Rally**（被弾後すぐ攻撃）
2. **Bonfire** 近接で受動回復
3. **Bandage**（30秒で900HP、被弾で中断）
4. **Potion**（Minor 35% / Standard 55%、即時）

## スタミナ管理

詳細は [体幹・スタミナ](poise-stamina.md) を参照。要点：

- 青 → 黄 → **赤点滅（Winded）** の3段階
- **Winded 状態 = スプリント・ドッジ不可の大ペナルティ**
- **20〜30%残した段階で攻撃を止める**のが基本

## 戦闘の基本的な流れ

1. **敵のパターンを観察**（最初の1〜2発を捌く）
2. **Heavyで体幹（シールド）を削る**
3. **シールド破壊 → スタガー中に集中攻撃**
4. **危険を感じたら即離脱してスタミナ回復**
5. **被弾したらRallyで即反撃してHPを取り戻す**

「**ヒット＆ラン**」が基本戦術。1〜2発当てて離脱する繰り返しで安全に削る。
