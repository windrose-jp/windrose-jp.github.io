## 💀 プレイヤー死亡時のリスポーン

### リスポーン地点の優先順位

| 優先 | 地点 | 条件 |
|------|------|------|
| **1** | **Tent**（最優先） | クラフト後**相互作用してアクティブ化が必須**（設置だけではダメ） |
| **2** | **Bonfire** | Tent が無い場合のデフォルト復活点 |
| **3** | **本拠地** | Tent も Bonfire も未設定の場合 |

> **「ベッド」は独立した復活ポイントとしては存在しない**。Tent が事実上の役割を兼ねる。

- **リスポーン待機**: 約3秒のディレイ後に復活
- 遠征時は**入口に必ず Tent をアクティブ化**してから突入する

### 死亡ペナルティ

| 項目 | 仕様 |
|------|------|
| **装備中の武器・防具** | **保持**（ロストしない） |
| **クイックバー枠（1〜4）** | **保持** |
| **バックパック中身（素材等）** | **全ロスト** → 死亡地点に「墓（tombstone）」が生成 |
| **マップ表示** | 赤い**墓アイコン**で死亡地点が表示 |
| **ワールド内表示** | **光るバスケット**として可視化 |
| **XP** | **ロストなし**、キャラ進行は完全保持 |
| **食事バフ** | **消滅**、再適用が必要 |
| **敵HP** | **死亡をまたいで回復しない** → 消耗戦（attrition）戦法が成立 |

> **戦術活用**: 雑魚敵は死亡をまたいでHPが維持されるため、**Tent から復活して再突撃で削り切る「死に特攻」が成立する**。ソウルライクだが寛容な仕様。

### Plague（疫病）による即死

**Wood Plague**（汚染木地帯）に長居して Plague ゲージが MAX になると**即死**。通常死と同等の処理（インベントリ→墓、装備保持）で**特殊ペナルティはない**が、再突入時の対策必須。

→ 詳細は [呪われた沼地のPlagueハザード](biomes.md#plague疫病ハザード) を参照

### ボスアリーナ内の特殊ルール

- ボスアリーナは **instanced（独立空間）** 扱い
- **内部に Tent 設置不可** → 死亡時はアリーナ外まで戻され**全行程やり直し**
- マルチプレイでは未参加プレイヤー専用に同ボス枠が再ロードされる

## ⛵ 船関連のリスポーン

### Ketch（スターター艇）の召喚仕様

| 項目 | 仕様 |
|------|------|
| 召喚キー | **K キー**（どこからでも召喚可能） |
| **Wharf からの完全修理リスポーン** | **Wood 20 を支払うと完全修理状態で Ketch がリスポーン**（Wharf 近接時） |
| 沈没時 | **Wharf でリカバリー可能** |
| Cargo の保護 | 沈没時も**Cargo Hold の素材は保持される** |

### 戦闘艦（Brig / Frigate）の沈没後

- **Wharf からのリカバリー必須**（Wood ×20 が基本）
- リカバリー費用は船サイズ・ティアでスケール
- 沈没した船は失われない（Wharf 経由で復元）

### 船上では直接リスポーンできない

- 死亡時に**船上でリスポーンする手段は存在しない**
- 船上で死ぬと最寄りの Tent / Bonfire / 本拠地に戻される
- **回避策**: 海岸に Tent を設置してから出航

> 情報源: [gamerblurb Ship Respawn](https://gamerblurb.com/articles/windrose-how-to-respawn-on-ship) / [method.gg Wharf](https://www.method.gg/windrose/the-wharf-in-windrose-how-to-build-it-what-it-does)

## 🔑 最重要: Bonfire（焚き火）半径は素材リスポーンをブロックする

**採掘拠点と生活拠点を分離するのは、Windrose 運用の絶対条件。**

| 項目 | 仕様 |
|------|------|
| 効果 | 焚き火の半径内では **鉱石・一部植物がリスポーンしない** |
| 対象素材 | Copper（銅）, Stone, Clay 等で確認。他の鉱石も同様と推定 |
| 解除方法 | **焚き火を撤去（解体）すれば該当エリアの素材サイクルが再開** |
| 半径の具体値 | **未公開**（英語ソースすべて定性表現のみ） |
| 影響範囲 | 焚き火は Respawn Anchor（プレイヤー復活点）・クラフトステーション動力・休息バフ源を兼ねるため、拠点に必要 |

### 運用方針
- 採掘地点（銅鉱脈など）の**近くに焚き火を置かない**
- クラフトステーションは焚き火半径内に置く必要があるため、**生活拠点は鉱脈から少し離す**
- 既に焚き火で詰んでいる場合は、焚き火を一旦撤去してリフレッシュを待つ

### Shrine（青い祠）でのボス再戦

ボスアリーナ内の**右側に青い光を放つ Shrine** があり、ここから再挑戦できる。「一度倒したら終わり」という誤認が多いが、何度でも戦える。

⚠️ **再戦時のドロップ仕様（一次情報未確認）**

複数の英語攻略サイト（mobalytics 系）で「**初回撃破時のみユニーク報酬がドロップ、2回目以降はドロップなし**」と記載されているが、**Steam 公式コミュニティや開発者発言での明示的な確認は取れていない**。一次情報での裏取りができていないため、本 wiki では参考情報として記載するに留める。

- 二次情報サイトでの主張: 初回のみドロップ、再戦は練習・ロアプレイ用
- マルチプレイ時には未参加プレイヤー枠で再ロードされる仕様あり（こちらは複数報告あり）
- Bosun（ミニボス）は別仕様で周回ファーム可能（こちらは確認済み）

> ⚠️ **「再戦で素材ファームしよう」と試みる前に、自分の環境で1〜2回試して挙動を確認することを推奨**。

> 出典: [Steam: Boss Respawn Discussion](https://steamcommunity.com/app/3041230/discussions/0/802345327968535603/)（実プレイヤー議論）

## 木・植物のリスポーン

### 木（通常の伐採対象）

**Steam 一次情報による正しい挙動**: 木は **Bonfire 半径外なら再生する、半径内では再生しない**（Bonfire 距離ベースの仕様）。樹種別の例外（「Foothills の Divi-Divi 木だけ復活する」等）は**英語攻略サイト（Deltia's Gaming 系）が独自に流布した可能性が高く、Steam 公式コミュニティの一次情報では確認できない**。

| 条件 | リスポーン |
|------|-----------|
| **Bonfire 半径外で伐採した木** | ✅ 再生する |
| **Bonfire 半径内で伐採した木** | ❌ 再生しない |
| Palm / Ficus / Banana（実が成る木） | 種から植え直し可能 |

> **重要：拠点の Bonfire 周囲の伐採は「使い捨て」になる**。建築のための大量伐採は Bonfire から離れた場所で行うこと。

- 「切り株（stump）を残せば再生する」説もあるが**未検証**
- 一般的な木の補給手段は **palm / ficus / banana の種を植え直す** こと

> 出典: [Steam Community 木のリスポーン議論（複数スレッド）](https://steamcommunity.com/app/3041230/discussions/0/802344996812569893/) — Bonfire 半径ベースの仕様が複数プレイヤーから独立に報告されている

> 情報源: [allthings.how Trees Grow Back](https://allthings.how/do-trees-grow-back-in-windrose-respawn-rules-explained/)

### ヤシの実・バナナ等（揺すって取るタイプ）
- **木を切らずに取る**（近接プロンプトで落とす方式）
- 再取得まで「長い」との報告（具体秒数不明）

### ハーブ・野菜類
- **Chili Pepper、Potato** は鉱石より早くリスポーン
- 島ごとに最低2か所ずつの固定畑あり
- **Cotton、Hemp** 等の繊維植物のリスポーンは英語ソース未確認

## 敵・ボスのリスポーン

### 雑魚敵
- **リスポーンする**
- **プレイヤー死亡→Tent復活で即戻ると、敵HPは維持されたまま**  
  → ソウルライクだが「死に特攻で削り切る」戦法が成立する

### 敵海賊船
- **non-quest の敵船はリスポーンする**
- quest 関連の敵船は別扱いの可能性

### 敵船クルー（ボーディング時）
- **無限湧き**
- 船を沈めてから最後の1隻だけボーディングする戦術が推奨
- バランス調整の議論対象

### ボス
- **アリーナ外の Shrine（祠）経由で再戦可能**  
  ※「一度限り」という誤認が多いが、再挑戦できる
- Thomas Richards（第1ボス）で確認。他ボスも同様と推定
- ボスアリーナ内は「instanced」扱いで **Tent 設置不可**

## POI・宝箱のリスポーン

### リスポーンしないもの（1回限り）
- **固定 POI Chest**（Smuggler's Cache, Abandoned Buccaneer Camp 等）— 警備敵も含めて回収後は再湧きなし
- **Buried Treasure（埋蔵金）** — 固定位置・固定戦利品、シャベルで掘ったら終わり
- **Quest オブジェクト**
- **砂浜の漂着物**（driftwood, seashells, seaweed, clams）

### リスポーンするもの
- **敵船の船上 loot**（船自体がリスポーンするため）
- 一般の陸上 chest（non-POI, non-quest）— Coop では再湧き報告あり、Solo では報告少（要検証）

### 豆知識
- POI chest の loot は「キャラクターID に紐づく」仕様で、**別キャラで同じ POI を訪れると再取得可能** との報告あり

### 海賊キャンプ周期（コミュニティ通説）

**海賊キャンプ（Pirate Camp POI）は一定周期でフルリセット**されると報告されている。複数の英語攻略サイト（adventuregamers / thegameswiki 等）が「**24 in-game 時間（1 in-game day）でリセット**」という具体数値で一致しているが、**WHOLF の Steam 投稿原文に該当数値の記載はなく、開発者発言での確認も取れていない**。プレイヤー検証ベースの通説として参考に留めること。

| リセット対象 | 内容 |
|------------|------|
| 敵海賊 | 全員復活 |
| チェスト | 中身が再生成 |
| Piled crates / barrels | 復活 |

- **キャンプを「全チェスト空・全敵討伐」してから一定時間放置** すると次サイクルで完全リセット
- 周回素材集めの最効率手段
- **Island 2 のキャンプ群が効率的な日次ファームループ**として知られる

> 情報源: [thegameswiki Pirate Camps](https://thegameswiki.com/windrose/wiki/pirate-camps) / Steam Community プレイヤー検証（**Officer WHOLF | BLACKBEARD はコミュニティモデレーターのボランティアであり開発者ではない点に注意**）

## リスポーントリガー

### ベッドで睡眠（Simple Bed）
- **時間スキップがメイン機能**（Q キーで使用）
- 鉱石リスポーン時間を加速する効果あり（xmodhub 報告）
- 開始時の Tent では不可、**屋根付き構造物内の Simple Bed が必要**
- 食事バフ・休息バフは維持される

### 島を離れて戻る
- 船で別の島へ行き戻ると素材が復活する報告多数
- 特に **Copper で有効**
- 同一島内で遠くに移動→戻るでも一定程度リスポーンする報告あり

### ゲーム内時間 vs 実時間
- **どちらが主トリガーかは未確定**
- 両方の報告がある（in-game day 説が主流だが、実時間で復活する報告も複数）
- オフライン時にタイマーが進むかは未確定

### ワールド再ロード
- メインメニューに戻って再ログインすると **素材が即リフレッシュ**
- Copper のリスポーン待ちをスキップする実用Tips
- 注意: 食事バフもリセットされる

### 参考: ゲーム内時間の体感
- 昼: 約 5〜10 分
- 夜: 最大 1 時間
- ※ 開発公式の数値は未公開、プレイヤー体感値

## マルチプレイ仕様

| 項目 | 仕様 |
|------|------|
| 素材ノード（Wood, Stone, Copper 等） | **ワールド共有**（先取り争い発生） |
| Chest の loot | **プレイヤー個別**（取り合い発生しない） |
| ワールドセーブ | **ホスト中心**（ホスト不在時プレイ不可） |
| Dedicated Server | Nitrado 経由で月 $10〜15 程度で借用可（常時稼働ワールド向け） |
| キャラクター進捗 | **全ワールド横断で保持**（装備・stat・インベントリ） |
| 推奨人数 | 最大 8 人だが **推奨 4 人**（パフォーマンス影響） |

### 8人規模での注意
- 1〜4人向けバランス調整されているため、**8人規模では Copper 等のリスポーンが追いつかない** 報告あり
- 先行プレイヤーが素材を取り切ると後続プレイヤーのチュートリアル進行が詰む報告あり（デモ時代からの問題、Early Access で残存の可能性）

以下は英語コミュニティでも確定していない項目です。プレイ中に判明した場合は情報源として記録推奨：

- Bonfire 半径の具体メートル数
- Bonfire 撤去後、素材リスポーンが即時か次サイクル待ちか
- Copper 以外の鉱石の正確なリスポーン日数
- 1 in-game 日 = 実時間何分の厳密値
- **In-game day / real-time どちらが主トリガーか**（最大の論点）
- オフライン時もリスポーンタイマーが進むか
- 魚・fishing node のリスポーン仕様
- Cotton, Hemp 等繊維植物のリスポーン
- Deer, Crocodile, Chicken 等動物の正確なリスポーン間隔
- Elite Mob の再湧き仕様
- Clay が完全非リスポーンか条件付きリスポーンか
- 木の切り株残し再生説の真偽
- 死亡墓（tombstone）に時間制限があるか
