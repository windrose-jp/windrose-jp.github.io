# リスポーン（再出現）仕様

> **注意**: Windrose は 2026年4月14日 Early Access 開始の新作で、リスポーン仕様は**未公式ドキュメント化**の領域が多く、ほぼ全てがコミュニティ検証情報です。本ページの内容は **2026年4月時点** の英語コミュニティ報告に基づきます。今後のパッチで変動する可能性があります。

> 情報源: Steam Community Discussions（複数スレッド）、xmodhub、tposegaming、allthings.how、adventuregamers、boostmatch、thegameswatch、PC Gamer、GameSpot 等の英語ソース

---

## 🔑 最重要: Bonfire（焚き火）半径は素材リスポーンをブロックする

**採掘拠点と生活拠点を分離するのは、Windrose 運用の絶対条件。**

| 項目 | 仕様 |
|------|------|
| 効果 | 焚き火の半径内では **鉱石・一部植物がリスポーンしない** |
| 対象素材 | Copper（銅）, Stone, Clay 等で確認。他の鉱石も同様と推定 |
| 解除方法 | **焚き火を撤去（解体）すれば該当エリアの素材サイクルが再開** |
| 半径の具体値 | **未公開**（英語ソースすべて定性表現のみ） |
| 影響範囲 | 焚き火は Respawn Anchor（プレイヤー復活点）・クラフトステーション動力・休息バフ源を兼ねるため、拠点に必要 |

### 運用方針
- 採掘地点（銅鉱脈など）の**近くに焚き火を置かない**
- クラフトステーションは焚き火半径内に置く必要があるため、**生活拠点は鉱脈から少し離す**
- 既に焚き火で詰んでいる場合は、焚き火を一旦撤去してリフレッシュを待つ

### Shrine（青い祠）でのボス再戦

ボスアリーナ内の**右側に青い光を放つ Shrine** があり、ここから再挑戦できる。「一度倒したら終わり」という誤認が多いが、何度でも戦える。報酬ドロップが初回と同等かどうかは未確認【要検証】。

## 鉱石のリスポーン詳細

### Copper（銅）— 最も報告の多い素材
- **リスポーン時間**: 報告に大きな幅あり
  - 「3〜5 in-game 日」説（主流）
  - 「6 in-game 時間」説
  - 「実時間でログアウト→翌日ログインで満タン復活」説
- **In-game時間 or 実時間** のどちらが主トリガーかは **未確定**
- **Bonfire 半径内では復活しない**（確定仕様）
- 実用Tips: **迷ったらワールド再ロード**（メインメニューに戻って再ログイン）で即リフレッシュされる

### その他の鉱石

| 素材 | リスポーン | 備考 |
|------|----------|------|
| Stone（石） | ✅ する | Bonfire 半径内は不可（推定） |
| Iron（鉄） | ✅ 3日周期の報告 | Foothills で入手、第1ボス撃破後にアンロック |
| Gold | ✅ する | 固定座標 |
| Clay（粘土） | ⚠️ 報告バラつき | ほぼ復活しない説が強い、少なくとも Bonfire 近くは確実にNG |
| Tin / Lead / Coal / Silver | ❓ 英語ソース未確認 | |

## 木・植物のリスポーン

### 木（通常の伐採対象）
- 報告にバラつきあり
- 「切り株（stump）を残せば再生する」説あり（未検証）
- 「完全伐採してもリスポーンした」報告もあり
- 建築地周辺で自然に木が湧くという報告 = リスポーン自体は起きている

### ヤシの実・バナナ等（揺すって取るタイプ）
- **木を切らずに取る**（近接プロンプトで落とす方式）
- 再取得まで「長い」との報告（具体秒数不明）

### ハーブ・野菜類
- **Chili Pepper、Potato** は鉱石より早くリスポーン
- 島ごとに最低2か所ずつの固定畑あり
- **Cotton、Hemp** 等の繊維植物のリスポーンは英語ソース未確認

## 敵・ボスのリスポーン

### 雑魚敵
- **リスポーンする**
- **プレイヤー死亡→Tent復活で即戻ると、敵HPは維持されたまま**  
  → ソウルライクだが「死に特攻で削り切る」戦法が成立する

### 敵海賊船
- **non-quest の敵船はリスポーンする**
- quest 関連の敵船は別扱いの可能性

### 敵船クルー（ボーディング時）
- **無限湧き**
- 船を沈めてから最後の1隻だけボーディングする戦術が推奨
- バランス調整の議論対象

### ボス
- **アリーナ外の Shrine（祠）経由で再戦可能**  
  ※「一度限り」という誤認が多いが、再挑戦できる
- Thomas Richards（第1ボス）で確認。他ボスも同様と推定
- ボスアリーナ内は「instanced」扱いで **Tent 設置不可**

## POI・宝箱のリスポーン

### リスポーンしないもの（1回限り）
- **固定 POI Chest**（Smuggler's Cache, Abandoned Buccaneer Camp 等）— 警備敵も含めて回収後は再湧きなし
- **Buried Treasure（埋蔵金）** — 固定位置・固定戦利品、シャベルで掘ったら終わり
- **Quest オブジェクト**
- **砂浜の漂着物**（driftwood, seashells, seaweed, clams）

### リスポーンするもの
- **敵船の船上 loot**（船自体がリスポーンするため）
- 一般の陸上 chest（non-POI, non-quest）— Coop では再湧き報告あり、Solo では報告少（要検証）

### 豆知識
- POI chest の loot は「キャラクターID に紐づく」仕様で、**別キャラで同じ POI を訪れると再取得可能** との報告あり

## リスポーントリガー

### ベッドで睡眠（Simple Bed）
- **時間スキップがメイン機能**（Q キーで使用）
- 鉱石リスポーン時間を加速する効果あり（xmodhub 報告）
- 開始時の Tent では不可、**屋根付き構造物内の Simple Bed が必要**
- 食事バフ・休息バフは維持される

### 島を離れて戻る
- 船で別の島へ行き戻ると素材が復活する報告多数
- 特に **Copper で有効**
- 同一島内で遠くに移動→戻るでも一定程度リスポーンする報告あり

### ゲーム内時間 vs 実時間
- **どちらが主トリガーかは未確定**
- 両方の報告がある（in-game day 説が主流だが、実時間で復活する報告も複数）
- オフライン時にタイマーが進むかは未確定

### ワールド再ロード
- メインメニューに戻って再ログインすると **素材が即リフレッシュ**
- Copper のリスポーン待ちをスキップする実用Tips
- 注意: 食事バフもリセットされる

### 参考: ゲーム内時間の体感
- 昼: 約 5〜10 分
- 夜: 最大 1 時間
- ※ 開発公式の数値は未公開、プレイヤー体感値

## マルチプレイ仕様

| 項目 | 仕様 |
|------|------|
| 素材ノード（Wood, Stone, Copper 等） | **ワールド共有**（先取り争い発生） |
| Chest の loot | **プレイヤー個別**（取り合い発生しない） |
| ワールドセーブ | **ホスト中心**（ホスト不在時プレイ不可） |
| Dedicated Server | Nitrado 経由で月 $10〜15 程度で借用可（常時稼働ワールド向け） |
| キャラクター進捗 | **全ワールド横断で保持**（装備・stat・インベントリ） |
| 推奨人数 | 最大 8 人だが **推奨 4 人**（パフォーマンス影響） |

### 8人規模での注意
- 1〜4人向けバランス調整されているため、**8人規模では Copper 等のリスポーンが追いつかない** 報告あり
- 先行プレイヤーが素材を取り切ると後続プレイヤーのチュートリアル進行が詰む報告あり（デモ時代からの問題、Early Access で残存の可能性）

## 未確定・要検証項目

- Bonfire 半径の具体メートル数
- Bonfire 撤去後、素材リスポーンが即時か次サイクル待ちか
- Copper 以外の鉱石の正確なリスポーン日数
- 1 in-game 日 = 実時間何分の厳密値
- In-game day / real-time どちらが主トリガーか
- オフライン時もリスポーンタイマーが進むか
- 魚・fishing node のリスポーン仕様
- Cotton, Hemp 等繊維植物のリスポーン
- Deer, Crocodile, Chicken 等動物の正確なリスポーン間隔
- Elite Mob の再湧き仕様
- Clay が完全非リスポーンか条件付きリスポーンか
- 木の切り株残し再生説の真偽
- ボス再戦時の報酬ドロップ仕様（初回と同じか）
