# ダンジョン・POI

> 情報源: [Steam ストアページ](https://store.steampowered.com/app/3041230/Windrose/)

## 概要

Windroseには**100以上のハンドクラフトされたポイント・オブ・インタレスト（POI）**が存在します。洞窟・遺跡・敵アジトなど様々な形式があり、探索や戦闘・報酬が得られます。

## POIの種類（確認中）

| 種類 | 内容 |
|------|------|
| ダンジョン | 敵が配置された探索エリア。ボスや宝が存在 |
| 遺跡 | 古代の遺構。ストーリーに関わるものも |
| 埋蔵金スポット | 赤い布の木が目印。シャベルで掘ると宝 |
| 敵アジト | 海賊や敵NPCの拠点 |

## 探索のコツ

- 島に上陸したら周囲を歩き回ってPOIを探す
- 赤い布がついた木は埋蔵金のサイン
- ファストトラベルベルを入り口付近に設置しておくと攻略が楽になる

各POIの詳細・ボス情報・報酬は情報収集中です。
