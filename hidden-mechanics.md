# ゲームが教えてくれない仕様

チュートリアルにも公式ドキュメントにも記載されていない、知っていると立ち回りが根本から変わる仕様をまとめています。

## 戦闘の隠し仕様

### 中クリックの特殊攻撃は武器ごとに異なる

「ヘビー攻撃＝中クリック」は初心者Tipsに記載されているが、**武器によってモーションがまったく異なる**点は説明されない。

| 武器種 | 中クリックの効果 |
|--------|----------------|
| 剣/サーベル系 | ヘビースラッシュ |
| **Rapier** | **2連続スラッスト（二段斬り）** |
| **Musket** | **精密射撃（倍率射撃）** |
| **Halberd** | 体幹大削りの薙ぎ払い |

同様に **F キー = Special Attack**（コントローラは LB+RB / L1+R1 同時押し）も未説明。武器ごとに固有効果がある。

### 武器種によってパリィが機能しない

すべての武器でパリィできると思いがちだが、**武器種によって適性が大きく異なる**。

| 武器種 | パリィ適性 |
|--------|-----------|
| Saber / Greatsword / Rapier | ✅ 有効 |
| Club / Mace | ⚠️ 効果小 |
| **Halberd** | **❌ ほぼ機能しない**（弱い攻撃のみ通る、代わりに体幹を消費） |

ハルバードを使う場合、パリィに頼らずヘビー攻撃での体幹削りを主軸にすること。

### 敵によって体幹ゲージの数が違う

| 敵 | Shield 数 |
|----|----------|
| Dodo / Sow（小型動物） | 2枚 |
| Boar（イノシシ） | 3枚（+スタン中に4発目で吹き飛ぶ） |
| Sturdy Halberd 海賊 | **4枚** |

Shieldが多い敵に対してライト攻撃を連打しても倒れない。パリィやヘビー攻撃でまとめて削るのが正解。

### Rally：薄色HPは攻撃で取り返せる

Bloodborne 式の仕様。被弾後、HPバーの一部が**薄い色で残る**（「取り返し可能なHP」）。

- **被弾直後に素早く攻撃を当てる**と薄色部分が回復する
- スタミナ管理に余裕があれば、被弾後の即反撃が回復手段になる
- 薄色部分は時間経過で消滅するので素早い判断が必要

### 攻撃モーションはキャンセルできない

ライト攻撃を2回振り始めたら、途中でキャンセルして回避に移れない。**攻撃を始めたら必ず後退**することが前提の設計。スタミナを赤にしてから回避しようとするパターンが最も危険。

### 銃リロード中は武器持ち替えが速い

銃のリロードアニメーションは約3秒かかる。リロードより**近接武器に持ち替えた方が速く次のアクションに移れる**。遠距離→近接→遠距離のサイクルが実戦的。

### 3×3フェンスで安全に削る

**Wood Fence を3×3に囲んで内側に敵を引き込む**と、動物・メレー敵は突破できない。内側から Rapier や Halberd で突いて安全に削れる。

## 拠点・建築の隠し仕様

### Comfort は「種類数」で決まる（量ではない）

拠点の Rested バフ持続時間を伸ばす **Comfort ゲージは、家具の「サブカテゴリごとに+1」**。同じ家具を10個置いても+1のまま。

- **9つのサブカテゴリから1種類ずつ**置けば Comfort 10 に到達
- 目安: 椅子・テーブル・棚・ランプ・ラグ・鏡・装飾品・絵画・トロフィー
- **トロフィー（動物の頭部）のみ例外**：個別加算で最大+12 Comfort まで積める

> Rested バフは快適度レベルが上がるごとに持続時間が延びる（最大 30 分）。快適度 Lv ごとに約+1分20秒。詳細は[食事・ポーション](character/buffs.md)を参照。

### 1つのアドオンで全 Workbench を一括アップグレード

**Sawhorse（アドオン）を1個設置するだけで、Bonfire 範囲内の全 Workbench が Lv2 になる**。Workbench ごとに個別設置は不要。Sawhorse は Workbench に隣接させること。

### 屋根の要/不要はステーションによって決まっている

| 屋根 **必須** | 屋根 **不可（屋外専用）** |
|-------------|------------------------|
| Armor Workbench | Charcoal Kiln（炭焼き窯） |
| Weaponsmith Workshop | Smelting Furnace / Large Smelting Furnace |
| Alchemy Table | Jewelry Table |
| Enchanting Table | |
| Millstone / Spinning Wheel | |
| Tanning Rig | |
| Shipwright's Workshop | |

Armor Workbench は「完全密閉の建物」でなく、**頭上に屋根があればOK**（壁なしでも機能する）。

### Workbench は Bonfire なしで設置・使用できる

**Workbench だけは Bonfire の暖気なしに単独で動作する**唯一のクラフトステーション。遠征先や拠点外での応急クラフトに活用できる。

### 倒した木が建築中の構造物にダメージを与える

伐採時、**倒れた木が近くの建造物にダメージ**を与える。拠点周辺で伐採する際は建物から距離を取るか、倒れる方向を確認すること。

### Disassembly Table で不要装備から素材を 100% 回収

装備を捨てる前に **Disassembly Table（解体台）を使うと基本素材を 100% 取り戻せる**。ただしアップグレードに使った素材は戻らない。装備のサイクルが成立するので不要装備は必ず解体すること。

## 船・海戦の隠し仕様

### 風向きは現在ゲームプレイに影響しない

マスト上の旗や空の雲で風向きを読めるが、**現バージョン（2026年4月）では風向きが船速に影響しない**。どの方向に向いても速度は一定。将来のアップデートで変更予定の可能性あり。

### 左舷・右舷・前方砲は独立したクールダウン

大砲は**左舷/右舷/前方の3系統が独立してリロードされる**。

- 左舷で撃った直後に旋回して右舷を向ければ即撃てる
- サイクル射撃（左→旋回→右→旋回→左）で継続砲撃が可能
- CD は砲の種類による（12ポンドで約11秒）

### Chain Shot（鎖弾）は帆・マスト専用

Chain Shot は**帆やマストを狙う弾**。船体に命中しても与ダメージは微小で効率が悪い。Chain Shot の命中で帆が裂けると、敵船に重い移動速度デバフが入る。逃げる船を追う場面で有効。

### 砲弾は高めに弧を描く——下を狙え

大砲の砲弾は**放物線軌道**のため、センターマスに照準を合わせると上を飛び越えてしまう。**目標の中央よりやや下**を狙うのが正確な当て方。また大波が弾を遮断するので、波の動きを見てから撃つこと。

### 船が沈んでも積み荷（Cargo）は保護される

船が撃沈されると個人インベントリのアイテムは失うリスクがあるが、**船のカーゴホールドに積んでいた素材は沈没後も保護される**。Wharf で船を修復・復旧するとそのまま取り出せる。高価な素材は個人インベに持たず Cargo に入れておくと安全。

### 操舵中でもキーでファストトラベルできる

**船を操舵している状態で K キー（船を呼ぶ）または M キー（マップを開く）からファストトラベルを実行できる**。わざわざ上陸しなくても、海上から直接テレポートが可能。

## システム・UIの隠し仕様

### ワールド選択画面の「EDIT」で難易度を後から変更できる

ワールド作成後でも、**ワールド選択画面の「EDIT」ボタンから敵のHP・ダメージ倍率を変更できる**。「難しすぎた」「簡単すぎた」と感じたら使える。ゲーム中のポーズメニューからは変更不可。

### Discovery タブを開かないとレシピに気づけない

素材を初めて入手すると発見通知が出るが、**G キーで Discovery タブを明示的に開かないと新レシピ解放に気づかない**。チュートリアルでもほぼ誘導されない。新しい素材を拾ったら必ず G キーを押す習慣をつけること。

### レシピは素材を一度拾わないと「?」のまま

**クラフトレシピは素材を1個でも手に入れて初めて開放される**。拠点に引きこもっているだけでは何も解放されない。序盤は積極的に島を探索して未知の素材に触れることがレシピ解放への近道。

### Talent リセットも無料・無制限

**ステータス（Attribute）・タレント（Talent）ともにリスペックは完全無料・何度でも可**。「タレントは慎重に選べ」という情報は誤り。詳細は[ステータス](character/stats.md)を参照。

### ホットバースロット 5–8 は船の消耗品専用

**ホットバーの 5〜8 番スロット**は通常の持ち物ではなく、**船に乗っているときに使う消耗品（Combat Repair Kit・Grog・食料等）の割り当て枠**。Wharf での船装備セット画面から設定する。

## 探索の隠し仕様

### Ketch 以降、操舵中にアイテムを自動拾得できる

**Ketch クラス以上の船で操舵しながら海面のアイテムに近づくと自動回収**される。スターターの小舟（Rowboat）では不可。Ketch 入手後は敵船撃沈後の漂流物回収が格段に楽になる。

### 停泊中、地図が自動的に埋まる

船を港や海岸に停めて放置している間も、**周辺エリアの地図が自動的に埋まっていく**（Cartography 進行）。意図的に留守にして地図を広げる運用も成立する。

### Shrine（青い祠）でボスに再挑戦できる

ボスアリーナの**右側にある「青い光を放つ Shrine（祠）**から、倒したボスに再挑戦できる。「一度倒したら終わり」という誤認が多いが、再戦可能。報酬ドロップが初回と同じかどうかは未確認【要検証】。

## リセット・リスポーンの隠し仕様

### リソースのリセットサイクル

ゲーム内の各リソースは異なる周期でリスポーンする。

| リソース | リスポーン周期 |
|---------|-------------|
| 鉱石・採掘ノード | **約6時間**（実時間 or in-game 日数、未確定） |
| 野菜・ハーブ・植物 | **約2時間** |
| 野生エネミー（動物・海賊） | **約2時間** |
| 海賊キャンプ（Camp POI） | **日次リセット**（全チェスト空化が条件） |
| **木（伐採済み）** | **再生しない**（切り株残しで再生説あり、未検証） |
| ボス（Shrine以外） | **自動リスポーンしない**（Shrine経由のみ再戦） |
| **Bosun（ミニボス系）** | **リスポーンする**（周回ファーム可能） |

詳細は[リスポーン仕様](exploration/respawn.md)を参照。

### Doctor Galen の無料ポーションは Bonfire ごとに独立

**Doctor Galen を複数の拠点に「定住」させると、各 Bonfire で独立したタイマーが走る**。複数島に設置してあれば各島で無料ポーションを受け取れる。詳細は[取り逃し注意集](missable.md#npc管理の罠galen--john-doe)を参照。
