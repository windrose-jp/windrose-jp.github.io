## 「ユニーク武器」とは

攻略コミュニティで「ユニーク武器」と呼ばれているのは、ゲーム内の独立したカテゴリではなく、**Epicレアリティに昇格（Ascension）した際に固有のパッシブ効果が付与された武器**のことです。

ゲーム内のレアリティ階層:

```
Common → Uncommon → Rare → Epic → (Legendary) → (Mythic)
```

- **Rare** まではどの武器でも通常強化（最大 Lv8）できる
- **Epic** に昇格するとパッシブ効果が解放され、最大 Lv15 まで強化可能
- Legendary / Mythic は概念が存在するが、武器カテゴリでの実装は現時点では確認されていない

---

## 通常武器との違い

| 比較項目 | 通常武器（Common〜Rare） | Epic武器（ユニーク） |
|---------|----------------------|------------------|
| パッシブ効果 | なし | **固有パッシブ1〜2個** |
| 強化上限 | Lv8 | **Lv15** |
| 昇格 | Rare → Epicへ昇格可能 | さらなる昇格で**第二パッシブ**も解放 |
| 見た目 | 通常 | 独自のビジュアル |
| 性能 | 同グレードのRareより低い | グレードが1段階上昇（Rare B → Epic A 等） |

**結論**: 単純な攻撃力もさることながら、**パッシブ効果の実用性**がユニーク武器の真の価値。「何%でドロップ」より「どのパッシブを得るか」で選ぶのが正しい。

## 入手方法（全パターン）

### 1. Ascension（昇格）— メインルート

Rare武器を **Lv8まで強化 → Weaponsmith Workshop で Tumbaga Ingot を使って Epic に昇格**。これが最も確実にEpic武器を手に入れる手段。

| 手順 | 内容 |
|------|------|
| 1 | 目当ての Rare 武器を入手 |
| 2 | Weaponsmith Workshop で Lv8 まで強化（Copper/Iron/Hide素材） |
| 3 | **Tumbaga Ingot** を Ascension 素材として使用 |
| 4 | Epic に昇格 → パッシブ効果が解放 |
| 5 | さらに Ascend すると **第二パッシブ**（Ascend追加効果）も解放 |

**Weaponsmith Workshop のグレードも必要**:
- Tier 2 = Anvil（鍛冶台）
- Tier 3 = Anvil + Bellows（鞴）

### 2. 固定宝箱（Rare状態での入手）

特定の固定ロケーションに Rare 武器が配置されている。Ascension すれば Epic に育てられる。

| 武器 | 場所 |
|------|------|
| **Rapier of a Thousand Cuts（Rare）** | 第1島の埋蔵金（**赤い旗がついた木の近くを掘る**） |

### 3. ボスドロップ

ボスを倒すと Epic 武器が直接報酬として得られる場合がある。

| ボス | 報酬武器 | 効果 |
|------|---------|------|
| **Ghost Captain** | Soul Eater | Agility+Vitality スケール、ヒットごとにライフスティール |
| Swamp Creature系 | Swamp Creature's Tooth | Cursed Swamps 専用の打撃武器 |

### 4. ファクション・バウンティ・探索

- バウンティハント（討伐依頼）の報酬
- 要塞強襲クリア報酬
- 軍事ファクション（評判システム）の報酬
- 第2島以降の高レベルエリアの宝箱

## Tumbaga Ingot（昇格素材）の入手

Ascension の核となる希少素材。**クラフト不可** — 固定入手のみ。

| 場所 | 内容 |
|------|------|
| **Ancient Ruins（古代遺跡）** | Rope 1本でラペル降下して進入、内部で入手 |
| **スターティングアイランドの隠し倉庫** | Abandoned Buccaneer Warehouse に隠しボタン・パズルあり（固定入手） |
| **Blackbeard's Treasure Map** | 宝の地図から Dig（掘る）で **Tumbaga Ingot ×3 + Epic アイテム ×1** |
| 宝箱 | 上位エリア・ダンジョンの宝箱から |

> 情報源: [Dexerto Tumbaga Ingot](https://www.dexerto.com/wikis/windrose/resources/tumbaga-ingot/) / [Destructoid Tumbaga](https://www.destructoid.com/how-to-get-tumbaga-ingots-in-windrose/)

### Ascension 運用の戦略

- 大きなパワースパイクになるため **メイン武器を最優先で Ascension**
- 複数武器を同時に育てず、1〜2本に集中投資
- 推奨 Ascension 例:
  - **Executioner（Halberd）** → キル時 +15% ダメージ軽減（Ward）
  - **Rapier of a Thousand Cuts** → Bleed スタック強化 + キル時転移
- ※ Ascension でのパッシブ選択をやり直せるかは情報不明（要検証）

## 昇格の戦略的な考え方

- **Rare以上のアイテムに強化素材を使う**。CommonやUncommonはEpicになれないため投資しない
- 同じ武器カテゴリでも **パッシブ次第でTierが変わる** ため、昇格前に効果を確認できるなら確認する
- Tumbaga Ingot は希少なため **本命ビルドの武器1本に絞って**先に Lv8 → Epic を目指す
- Epic Lv15 まで育てた武器は長期間主力になり得る
