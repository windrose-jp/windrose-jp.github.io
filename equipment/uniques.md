# ユニーク装備

> 情報源: [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/)

ユニーク装備は特殊な効果を持つ強力な武器・防具です。

## ユニーク武器

### 千の刃のレイピア（Rapier of a Thousand Cuts）

| 項目 | 内容 |
|------|------|
| 種別 | レイピア（近接・突き） |
| 特殊効果 | 攻撃時に出血（Bleed）を付与。最大5スタックまで積み重ね可 |
| 強み | スタック上限まで積むことで継続ダメージ（DoT）が非常に強力 |
| 入手方法 | 情報収集中 |

**使い方**: 連続攻撃で出血を素早く5スタックに乗せ、DoTで削るブリードビルドの核となる武器です。アップグレードするほどブリードのスケーリングが伸びます。

---

### エクゼキューショナー（Executioner）

| 項目 | 内容 |
|------|------|
| 種別 | ハルバード（近接・長柄） |
| 特殊効果 | 敵をキルするたびにクリティカル確率が上昇 |
| 強み | 雑魚敵の集団で真価を発揮。ハルバードの広範囲と組み合わせて連鎖キルを狙う |
| 入手方法 | 情報収集中 |

---

### 信頼のマスケット（Reliable Musket）

| 項目 | 内容 |
|------|------|
| 種別 | マスケット（遠距離） |
| 特殊効果 | 高火力 + 通常マスケットより速いリロード |
| 強み | マスケットの弱点である「リロードの遅さ」を補いつつ高火力を維持 |
| 入手方法 | 情報収集中 |
