# 防具一覧

> 情報源: [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/) / [Boostmatch Best Armor Sets](https://boostmatch.gg/blog/windrose/articles/windrose-best-armor-sets-builds-guide) / [Windrose.tools](https://windrose.tools/)

## 防具セットの基本

Windroseの防具は通常 **5部位セット**（頭・胴・手・足・脚）で構成されます。部位ごとに装備でき、**2pcボーナス / 4pcボーナス** のような段階的なセット効果を持つものが多いです。

序盤は **Survivor Set** で生存性を確保し、中盤以降は戦闘スタイルに応じて専用セットに移行するのが定石です。

---

## 確認済みの防具セット（全7セット）

### サバイバーセット（Survivor Set）

| 項目 | 内容 |
|------|------|
| レアリティ | Uncommon |
| 部位 | Bandana / Boots / Gloves / Jacket / Pants（5部位） |
| 特徴 | 軽装、序盤の標準装備 |
| 入手方法 | Doctor Galen、補給箱、基本商人から入手 |

**解説**: チュートリアル直後に確保できる序盤装備。特別なボーナスはないが軽量で動きやすい。次のセットが整うまでの繋ぎとして十分機能する。

---

### フリビュスティエセット（Flibustier Set）

| 項目 | 内容 |
|------|------|
| レアリティ | Rare |
| 部位 | Boots / Gloves / Hat / Jacket / Pants（5部位） |
| 4pcボーナス | **片手武器ダメージ +15%** / 攻撃スタミナ消費軽減 |
| 向いているビルド | Fencer系・片手武器・出血ビルド |

**解説**: 片手武器の火力と燃費を両立するセット。レイピア・サーベル・カットラス等のDPS重視ビルドで最適解になりやすい。**片手武器で攻めるなら最優先の目標セット**。

---

### パイクマンセット（Pikeman Set）

| 項目 | 内容 |
|------|------|
| レアリティ | Rare |
| 部位 | Boots / Cuirass / Gloves / Helmet / Pants（5部位） |
| 4pcボーナス | **両手武器ダメージ +15% / 最大HP +200** |
| 向いているビルド | Crusher系・両手剣・ハルバード |

**解説**: 両手武器の火力とHP上乗せを両立するタンクDPSセット。グレートソードやハルバードで正面から殴り合うビルドで真価を発揮。

---

### マークスマンセット（Marksman Set）

| 項目 | 内容 |
|------|------|
| レアリティ | Rare |
| 部位 | Boots / Doublet / Gloves / Pants / Tricorn（5部位） |
| 4pcボーナス | **遠距離ダメージ +15%** |
| 向いているビルド | Marksman系・銃器特化 |

**解説**: ピストル・マスケット・ブランダーバス主体のビルド向け。ダッシュ・スプリントのスタミナ消費を下げるバージョンも確認されており、ヒット＆ラン戦法と相性が良い。

---

### プリヴァティアセット（Privateer Set）

| 項目 | 内容 |
|------|------|
| レアリティ | Rare |
| 部位 | Boots / Doublet / Gloves / Hat / Pants（5部位） |
| 特徴 | 攻撃系混成ビルド向けのバランスセット |
| 入手方法 | 情報収集中 |

**解説**: 近接と遠距離を両方こなすハイブリッド型ビルドの土台として使われる。

---

### コンキスタドールセット（Conquistador Set）

| 項目 | 内容 |
|------|------|
| レアリティ | Rare |
| 部位 | Boots / Cuirass / Gloves / Helmet / Pants（5部位） |
| 4pcボーナス | **Bulwark**: 被弾しても攻撃モーションがよろけない |
| 向いているビルド | 重装タンク・正面から受け止めるビルド |

**解説**: **ボス戦や混戦で真価を発揮する防御特化セット**。攻撃中に敵の攻撃が来てもモーションがキャンセルされないため、確実にダメージを通せる。Co-opでのタンク役に最適。

---

### トラッカーセット（Tracker Set）

| 項目 | 内容 |
|------|------|
| レアリティ | Rare |
| 部位 | Boots / Gloves / Hat / Jacket / Pants（5部位） |
| 2pcボーナス | 生存性強化 |
| 向いているビルド | 攻撃系セットとの混成で生存性確保 |

**解説**: 2pcだけでも効果があるため、他のセット（Flibustier / Marksman等）と **混成** して生存性を底上げする使い方が定番。
