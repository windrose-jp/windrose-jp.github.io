## 敵の分類

Windroseの敵はバイオームや場所によって異なります。野生動物から人型の敵、そして強力なボスまで多様な敵が登場します。

## 共通の戦闘メカニクス

- 全ての敵が**ポイズ（Poise）**シールドを持つ
- ポイズを削ることで転倒させて大ダメージのチャンス
- バイオームが違うと敵の強さも変わる
