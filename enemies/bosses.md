# ボス攻略

> 情報源: [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/) / [Boostmatch ボス攻略ガイド](https://boostmatch.gg/blog/windrose/articles/how-to-beat-thomas-richards-windrose-boss-guide) / [allthings.how Windrose ガイド](https://allthings.how/)

## ボス攻略の共通原則

1. **Guard Shield（体幹ゲージ）を最初に破壊する** — 全ボス共通の前提。体幹を削り切らないとダメージがほとんど入らない
2. **Heavy Attack で体幹を削る** — Light Attack は体幹削り効率が悪い。重攻撃主体で戦う
3. **スタミナを青に保つ** — 赤になると回復に大ペナルティ。常にスタミナを意識
4. **パターンを観察してから攻撃** — 最初は1〜2回様子を見て、攻撃パターンを把握してから攻めに転じる
5. **ヒット＆ランを徹底** — 安全な時だけ攻撃し、危険を感じたら即離脱
6. **遠距離武器はボス用に温存** — 雑魚に弾薬を使わず、体幹ゲージ破壊後の集中火力として使う

## 進行の仕組み

各バイオームにボスが存在し、**ボスを倒すことで次のバイオームのギア製作・強化が解放**される。

| バイオーム | ギアレベル | 解放条件 |
|-----------|-----------|---------|
| 海岸のジャングル（Coastal Jungle） | Lv 1〜5 | スタート |
| 山麓（Foothills） | Lv 6〜10 | Coastal ボス撃破 |
| 呪われた沼地（Cursed Swamps） | Lv 11〜15 | Foothills ボス撃破 |

次のボスはゲーム内の**Discoveryタブ**で前ボス撃破後に解放される。

## ボスの再戦について

**ボスは一度倒しても再戦可能**（英語コミュニティ報告）。

- ボスアリーナ外の **Shrine（祠）** 経由で再挑戦できる
- 「一度限り」という誤認が多いが、練習や再チャレンジが可能
- 初回撃破でストーリーは進行するため、進行上の不可逆ポイントは踏む前提
- ※ ボスアリーナ内は「instanced」扱いで **Tent 設置不可**

詳細なリスポーン仕様は [リスポーン仕様](../exploration/respawn.md) を参照。

### トーマス・リチャーズ（Thomas Richards）

**クエスト**: 「Revenge is Best Served Cold（仇は冷めてから取れ）」

Coastal Jungle の最初のボス。序盤の実力試しとなる戦闘。

#### 基本情報

| 項目 | 内容 |
|------|------|
| バイオーム | Coastal Jungle |
| 推奨ギアレベル | Lv 3〜5 |
| 主な攻撃 | **2連斬り（固定パターン）** |
| 特殊機構 | Guard Shield（体幹ゲージ）— 破壊が攻略の前提 |

#### Guard Shield（体幹ゲージ）の仕組み

体力バーの下に**Guard Shield（ガードシールド）**が表示されている。これがある間はダメージが極端に低くなる。

- **Heavy Attack が最も効率よく体幹を削れる**
- Light Attack は体幹削り量が少なく非効率
- パリィも体幹削りに有効
- 体幹が0になると**ボスが転倒・よろける** → 大ダメージのチャンス

#### 攻略手順

1. **距離を取り、2連斬りのパターンを観察する**
2. 2連斬りの2発目が終わったタイミングで接近
3. **Heavy Attack で体幹を削る**（2〜3発）
4. スタミナが減ったら即離脱
5. 体幹が0になったら全力で攻撃（銃もここで使う）
6. Guard Shield が回復したら手順3に戻る

#### 推奨装備・戦術

| 項目 | 推奨 |
|------|------|
| 推奨武器 | **グレートソード**（リーチ・Strength 重攻撃・ブロック時のダメージ軽減） |
| 遠距離 | 体幹ゲージ破壊直後に使用。雑魚戦で弾を使い切らないこと |
| 防具 | できるだけ Rare 防具を揃えてから挑む |
| 食事バフ | 2スロット両方埋めて挑む |
| ポーション | Doctor Galen から当日分1本を必ず受け取ってから挑む |

#### Co-op での戦い方

| 役割 | 内容 |
|------|------|
| タンク（1名） | 正面で2連斬りを受け止め続ける |
| DPS（残り） | 側面・背後から Heavy Attack で体幹を削る |
| タイミング | 体幹破壊後に全員で集中火力 |

Co-op では役割分担で体幹破壊が格段に速くなる。

#### ドロップ・報酬

- ボス撃破後に**Foothills のギアレベル（Lv6〜）の製作・強化が解放**
- 具体的なドロップアイテムは情報収集中

## Foothills のボス

### イスラエル・ハンズ（Israel Hands）

**クエスト**: 「Needle in a Haystack」の到達点。Foothills 西端の神殿（Temple）に出現する第2ボス。

#### 基本情報

| 項目 | 内容 |
|------|------|
| バイオーム | 山麓（Foothills） |
| 推奨キャラLv | **9〜10** |
| 推奨ギアレベル | Lv 6〜10（Rare装備推奨） |
| アリーナ到達条件 | **Temple Keys** を北側の死んだ海賊から入手 |
| 特徴 | **幽霊系能力で遠近両用、攻撃が多彩** |

#### アリーナ構造

- Foothills 西端の神殿
- **アリーナ直前にファストトラベル地点**あり → 補給・Bonfire 再バフ可能
- ボス部屋は開けた空間で回避しやすい

#### 攻撃パターン（5種）

| # | 攻撃名 | テル（予兆） | 回避法 |
|---|--------|------------|--------|
| 1 | **Spectral Halberd Charge** | **赤い閃光**と共に突進 | **斜め横にダッシュ**。軌跡に毒のトレイルが残るので踏まない |
| 2 | **Overhead Swing → 多段コンボ** | 遅いワインドアップ（振り上げ） | **後退して meleeレンジ外へ**。パリィ非推奨（コンボが長い） |
| 3 | **Spectral Shot** | 肩の幽霊がピストルを構える | 構えた瞬間に**横ダッシュ** |
| 4 | **Spectral Whirl** | 骸骨が**赤刃2本**で出現 | **後方に1回ダッシュ**で射程外へ |
| 5 | **Soul Barrage** | 頭蓋骨群が空から降下 | **片側に連続スプリント**（1ダッジでは不足） |

#### 核心戦略

- **パリィより回避優先**（一般プレイヤー向け）
- Hands の攻撃後には**短いポーズ**がある → ここが攻撃窓
- **1〜2発の重打撃 → すぐ後退**が基本
- **コンボを繋げようとすると次の攻撃で確定被弾**する
- 赤エフェクトの攻撃は全てドッジで回避（ブロック・パリィ不可）

#### 推奨装備・消耗品

| 項目 | 推奨 |
|------|------|
| 武器 | **両手武器・重武器**（Greatsword / Halberd） |
| 料理 | **Spicy "Chicken" with Sweet Potato**（Vitality バフ）等 Rare料理で30分バフ |
| 回復 | **Healing Potion**（Bandage は被弾で切れるのでボス戦では非推奨） |
| 防具 | Rare セットボーナス（Pirate Camp襲撃 or Tortuga Faction Provisioner で設計図入手） |

#### ドロップ・報酬

- **Charon's Obol（アーティファクト）** — ボスドロップ
- **Soul Eater**（Rare武器、グレートソード）— 付近のチェストから入手可能
- 撃破で Cursed Swamps への進行が解放
