## ボス攻略の共通原則

1. **Guard Shield（体幹ゲージ）を最初に破壊する** — 全ボス共通の前提。体幹を削り切らないとダメージがほとんど入らない
2. **Heavy Attack で体幹を削る** — Light Attack は体幹削り効率が悪い。重攻撃主体で戦う
3. **スタミナを青に保つ** — 赤になると回復に大ペナルティ。常にスタミナを意識
4. **パターンを観察してから攻撃** — 最初は1〜2回様子を見て、攻撃パターンを把握してから攻めに転じる
5. **ヒット＆ランを徹底** — 安全な時だけ攻撃し、危険を感じたら即離脱
6. **遠距離武器はボス用に温存** — 雑魚に弾薬を使わず、体幹ゲージ破壊後の集中火力として使う

## 進行の仕組み

各バイオームにボスが存在し、**ボスを倒すことで次のバイオームのギア製作・強化が解放**される。

| バイオーム | ギアレベル | 解放条件 |
|-----------|-----------|---------|
| 海岸のジャングル（Coastal Jungle） | Lv 1〜5 | スタート |
| 山麓（Foothills） | Lv 6〜10 | Coastal ボス撃破 |
| 呪われた沼地（Cursed Swamps） | Lv 11〜15 | Foothills ボス撃破 |

次のボスはゲーム内の**Discoveryタブ**で前ボス撃破後に解放される。

## ボスの再戦について

**ボスは一度倒しても再戦可能**（英語コミュニティ報告）。

- ボスアリーナ外の **Shrine（祠）** 経由で再挑戦できる
- 「一度限り」という誤認が多いが、練習や再チャレンジが可能
- 初回撃破でストーリーは進行するため、進行上の不可逆ポイントは踏む前提
- ※ ボスアリーナ内は「instanced」扱いで **Tent 設置不可**

詳細なリスポーン仕様は [リスポーン仕様](../exploration/respawn.md) を参照。

Coastal Jungle の**海賊キャンプや難破船周辺に出現するフィールドミニボス**。Thomas Richards より先に多くのプレイヤーが遭遇し、序盤最大の壁になっている。

### トーマス・リチャーズ（Thomas Richards）

**クエスト**: 「Revenge is Best Served Cold（仇は冷めてから取れ）」
**場所**: Coastal Jungle 奥の木製の門を越えたキャンプ

Coastal Jungle の第1メインボス。「ソロ不可能」という声が多いが、攻撃パターンを把握すれば一定のリズムで戦える。

> 情報源: [method.gg Thomas Richards 攻略](https://www.method.gg/windrose/how-to-beat-thomas-richards-in-windrose-revenge-is-best-served-cold) / [BisectHosting 攻略ガイド](https://www.bisecthosting.com/blog/windrose-thomas-richards-coastal-jungle-boss-location-tips-tricks-rewards) / [keengamer 攻略](https://www.keengamer.com/articles/guides/how-to-beat-thomas-richards-in-windrose/)

#### 基本情報

| 項目 | 内容 |
|------|------|
| バイオーム | Coastal Jungle |
| 推奨ギアレベル | **Lv5（全装備Lv5推奨）** |
| 武器 | 斧 |
| 特殊機構 | Guard Shield（体幹ゲージ）— 破壊が攻略の前提 |

#### 攻撃パターン

| 攻撃 | 予兆（Tell） | 対処法 |
|------|-------------|--------|
| **2連撃コンボ** | 斧を2回振る（小さな間あり） | 2発目終わりに接近してカウンター。パリィ成功でスタン→追撃4〜5発可 |
| **3連撃コンボ** | 2連撃の後に前進しながら重撃を追加 | 2発目のタイミングで離脱し3発目を空振りさせる |
| **頭上叩きつけ（赤光）** | 斧を大きく振りかぶる＋赤いエフェクト | **横へ回避必須**。ブロック・パリィ不可 |
| **突進掴み（赤光）** | 前方に突進しながら赤いエフェクト | **横へ2回連続ダッシュ**。後退は危険（範囲に入る） |
| **爆弾投擲** | ボスが後退してグレネードを投げる | **ボスの背後に回り込む**（真横・後方は爆発に巻き込まれる） |

> **体力減少でのフェーズ変化**: 明確なフェーズ切替なし。HP低下で2連撃の速度・威力が上がる程度。

#### 核心戦略

**Guard Shield を削って「スタガー→大ダメージ」を繰り返すのが基本。**

1. 距離を取り、2連撃パターンを観察する（最初は攻撃しない）
2. 2発目が終わったタイミングで接近し **Heavy Attack×2〜3**
3. スタミナが減ったら即離脱
4. Guard Shield が0になったら**全火力集中**（遠距離武器もここで使う）
5. Guard Shield が回復したら手順2に戻る

> **欲張り禁止**: 3発目を狙うと突進掴みで確定被弾。2発打ったら必ず引く。

#### 推奨装備・バフ

| 項目 | 推奨 |
|------|------|
| 近接武器 | **グレートソード**（体幹削りS、リーチあり）/ **ラッパ銃**（遠距離から体幹削り可） |
| 防具 | **Pikeman Set×2 + Conquistador's Set×2**（HP+160、ダメ軽減+15%） |
| 食事 | 2スロット両方埋める（スタミナ補強+HP増加系が安定） |
| ポーション | **ヒーリングポーション**優先（包帯は被弾で中断するため非推奨） |
| 出発前バフ | **Bonfire の Rested バフ必須**。スタミナが別物になる |

#### Co-op での戦い方

| 役割 | 内容 |
|------|------|
| タンク（1名） | 正面で2連撃を受け、アグロを維持 |
| DPS（残り） | 側面・背後から Heavy Attack で体幹を削る |
| タイミング | 体幹破壊後に全員集中攻撃 |

## Foothills のボス

### イスラエル・ハンズ（Israel Hands）

**クエスト**: 「Needle in a Haystack」の到達点。Foothills 西端の神殿（Temple）に出現する第2ボス。

#### 基本情報

| 項目 | 内容 |
|------|------|
| バイオーム | 山麓（Foothills） |
| 推奨キャラLv | **9〜10** |
| 推奨ギアレベル | Lv 6〜10（Rare装備推奨） |
| アリーナ到達条件 | **Temple Keys** を北側の死んだ海賊から入手 |
| 特徴 | **幽霊系能力で遠近両用、攻撃が多彩** |

#### アリーナ構造

- Foothills 西端の神殿
- **アリーナ直前にファストトラベル地点**あり → 補給・Bonfire 再バフ可能
- ボス部屋は開けた空間で回避しやすい

#### 攻撃パターン（5種）

| # | 攻撃名 | テル（予兆） | 回避法 |
|---|--------|------------|--------|
| 1 | **Poison Dash（Spectral Halberd Charge）** | **赤い閃光**と共に突進 | **斜め横にダッシュ**。**軌跡に毒のトレイルが残る**ので絶対に踏まない（ヒットしなくても毒で削れる） |
| 2 | **Overhead Swing → 多段コンボ** | 遅いワインドアップ（振り上げ） | **後退して meleeレンジ外へ**。パリィ非推奨（コンボが長い） |
| 3 | **Spirit Cannon（Spectral Shot）** | 肩の幽霊がピストルを構える＋**音 cue** | **大きく弧を描いて全力スプリント**で完全回避（横ダッシュ1回では避けきれない） |
| 4 | **Whirlwind（Spectral Whirl）** | 骸骨が**赤刃2本**で出現＋赤グロー | **バックダッシュ**で射程外へ |
| 5 | **Soul Barrage** | 頭蓋骨群が空から降下、**浮上モーション** | **片側に連続スプリント**（1ダッジでは不足、走り続ける） |
| 6 | **Spectral Charge（red, hyper-armor）** | 赤グロー＋ハイパーアーマー突進 | **横ではなく通り抜けるダッシュ**（横回避だと当たる） |

#### 核心戦略

- **パリィより回避優先**（一般プレイヤー向け）
- Hands の攻撃後には**短いポーズ**がある → ここが攻撃窓
- **1〜2発の重打撃 → すぐ後退**が基本
- **コンボを繋げようとすると次の攻撃で確定被弾**する
- 赤エフェクトの攻撃は全てドッジで回避（ブロック・パリィ不可）

#### 推奨装備・消耗品

| 項目 | 推奨 |
|------|------|
| 武器 | **両手武器・重武器**（Greatsword / Halberd）または **Epic Rapier**（Bleed 5 スタック狙い） |
| 料理 | **Spicy "Chicken" with Sweet Potato**（Vitality バフ）+ **Coffee**（END+20） |
| 回復 | **Healing Potion**（Bandage は被弾で切れるのでボス戦では非推奨） |
| 防具 | **Conquistador 4部位**（被ダメ -15%・Bulwarkでよろけ無効） |

#### 戦闘ペット **Truffle** でアグロ分散

**Ancient Village の儀式で取得できる戦闘ペット「Truffle」**（装甲豚）を召喚すると、Israel Hands のアグロを分散できる。Truffle はヘビーアーマーで生存しやすく、攻撃の合間にプレイヤーが安全に削れる。

→ Truffle の取得方法は [取り逃し注意集](../missable.md#-truffle戦闘ペット装甲豚) を参照

#### ドロップ・報酬

- **Charon's Obol（アーティファクト）** — ボスドロップ・メイン進行必須
- **Soul Eater Greatsword**（Rare） — 付近のチェストから入手（life drain は Vitality スケール）
- **Undead Essence ×20〜30** — エンチャント素材
- **Ancient Chest** — 周辺
- 撃破で Cursed Swamps への進行が解放

## Cursed Swamps のボス

### High Priestess（ハイプリーステス）

**クエスト**: 「Forgotten Relics」最終段階
**場所**: Cursed Swamps 深部

Cursed Swamps の最終ボス（2026年4月 Early Access 時点）。**膿胞（Pustule）を破壊して弱点を露出させる**フェーズ移行がカギ。

> 情報源: [Deltia's Gaming 攻略](https://deltiasgaming.com/windrose-how-to-defeat-high-priestess/) / [BisectHosting 攻略](https://www.bisecthosting.com/blog/windrose-high-priestess-boss-cursed-swamps-location-tips-tricks-rewards) / [Sportskeeda 攻略](https://www.sportskeeda.com/esports/how-defeat-high-priestess-windrose)

#### 基本情報

| 項目 | 内容 |
|------|------|
| バイオーム | 呪われた沼地（Cursed Swamps） |
| 推奨ギアレベル | **Lv 11〜15（全装備Lv5必須）** |
| エリアの特徴 | ピンクに光るキノコ・森に触れると **Plague 感染** |
| 周辺の敵 | Plague Thrall / Plague Crocodile |

> ⚠️ **名称について**: 英語ソースで「Tainted Priestess」と「High Priestess」の両表記が混在している。ゲーム内の正式名称は「High Priestess」が有力。

#### 攻撃パターン

| 攻撃名 | 予兆（Tell） | 対処法 |
|--------|-------------|--------|
| **Giant Lunge（大跳躍）** | 後脚で溜め姿勢 | **横ダッジ必須**（ブロック不可） |
| **Tainted Spit（汚染唾液）** | 口を開けて前傾 | 1〜3球を個別ダッジ。Tainted ゲージに注意 |
| **Heavy Stomp（踏みつけ）** | 片足を高く上げる | 横スライドで回避 |
| **Roll-over（体当たり）** | 横に傾いて全身を倒す | **大きくダッジ×2**が必要 |
| **Forward Charge（突進）** | 正面を向いて助走 | 通過後に背面が短時間露出 → 攻撃チャンス |
| **Tongue/Tail Slam** | 尾または舌がぶれる | 見た目より判定が広い。早めのダッジ |

#### 核心戦略：膿胞→弱点露出ループ

1. 体表に**黄色く光る膿胞（Pustule）**が定期的に出現する
2. 膿胞を**速射ピストル**等で素早く破壊（マスケットは遅すぎる）
3. 膿胞を一定数破壊すると**脆弱フェーズ（Vulnerability Phase）**に移行 → 口内の弱点が露出
4. 弱点に**Heavy Attack×2〜3**を集中 → これが主要ダメージ源
5. 脆弱フェーズ終了後は手順1に戻る

> 欲張り禁止。弱点が開く時間は短く、3発目を狙うと掴み攻撃を食らう。**1〜2発→即ダッジ**を徹底。

#### 推奨装備・バフ

| 項目 | 推奨 |
|------|------|
| 遠距離 | **Drake's Double-Barreled Pistol 等の速射ピストル**（膿胞破壊用） |
| 近接 | Rare 以上の重武器（Greatsword / Halberd） |
| 食事 | バフ2スロット満杯 + Rested バフ必須 |
| スタミナ | 常に青色維持が最重要。弱点フェーズに備えて温存 |

#### ドロップ・報酬

| 報酬 | 内容 |
|------|------|
| **Charon's Obol** | メインストーリー進行必須アイテム |
| Tear of Sorrow ×10 | 素材 |
| XP 50 | 経験値 |
| **Soul Eater**（Epic） | ボス部屋の Ancient Chest から入手可能 |

Cursed Swamps の海上で遭遇する**世界ボス**。

| 項目 | 内容 |
|------|------|
| 座標 | **X: 150, Y: -300**（南海岸線） |
| マップ表示 | **巨大なスカルアイコン** |

| 準備 | 入念な装備とクルー推奨（具体的な攻撃パターン情報は現状少ない） |

### 攻略 Tip

- **Frigate 推奨**（火力・耐久ともに必要）
- **Combat Repair Kit を 5〜10 個常備**、修理クルー1〜2名常駐
- **Inkプール（黒い池）に入らない**: 視界遮断＋継続ダメ
- **触手はパリィ可** — タイミング合わせで反撃窓
- **帆を半分畳む**と被弾ダメ -30%（嵐イベント仕様）

## Chick Chan（Plague Crusher）— Stargazer 連動ボス

**Stargazer Tower 3基をクリアした後**に発生する Cursed Swamps 中央部のボス。「Fate of the Prophets」クエストの最終戦闘の1つ。

> 情報源: [neonlightsmedia Stargazer](https://www.neonlightsmedia.com/blog/windrose-stargazer-tower-puzzle-solution)

| 項目 | 内容 |
|------|------|
| 場所 | **Warlord's Death Site**（Cursed Swamps 中央部） |
| 推奨ギアレベル | Lv 11〜15 |
| 種別 | Plague 系大型 |
| 前提 | Stargazer Tower 3基すべて Solve 済み |

### 攻撃パターン

- 体に**発光する膿包（Pustule）**が定期的に出現
- 膿包をレンジ武器で破壊 → ポスチャー崩壊 → 接近コンボの DPS 窓
- High Priestess 同様の「膿包→弱点露出ループ」型

### 推奨装備

- **遠距離武器**（速射ピストル / Blunderbuss）で膿包破壊
- 近接は Halberd / Greatsword
- **Anti-plague consumable** 携行

## Ghost Captain（ゴーストキャプテン）— オプションボス

**Israel Hands とは別の独立したボス**。同じ「Needle in a Haystack」クエストライン周辺に存在するため混同されやすい。

> 情報源: [BoostMatch Ghost Captain 攻略](https://boostmatch.gg/blog/windrose/articles/windrose-ghost-captain-boss-fight-guide) / [The Games Wiki](https://thegameswiki.com/windrose/wiki/ghost-captain)

| 項目 | 内容 |
|------|------|
| 場所 | **Temple Dungeon**（インスタンスダンジョン） |
| 必須か | **任意**（オプションボス） |
| 推奨レベル | **10〜12** |
| クエスト | Needle in a Haystack の派生ブランチ |

#### 攻撃パターン

- 基本はThomas Richards と同じソウルライクメカニクスだが HP が多い
- **HP 約70%でスペクトル投射物**（追尾性あり・着弾時爆発）を通常攻撃にミックス
- 対処法：**横方向に弧を描くストレイフ** → 追尾コーンを外して自然消滅させる

#### 推奨戦術

- 武器：**Saber または Rapier**（短いコンボ後のパニッシュウィンドウを活用）。Rapier of a Thousand Cuts の出血スタックが特に有効
- 防具：**Conquistador's Set 4 ピース完全装備**で Bulwark（行動中断不可）が発動 → 攻撃の振り中・ポーション飲み中もよろけない。Ghost Captain の短い反撃窓を活かしつつ被弾リスクを大幅に削減できる（詳細は [防具一覧](../equipment/armor.md#コンキスタドールセットconquistador-set) 参照）
- 遠距離武器：Musket ビルドも有効。Ghost Captain は距離を取る攻撃モーションが多く、その間は安全に射撃できる
- 回復アイテムを多めに持参（Healing Food 15〜20 個推奨）
- ダンジョン入口付近にテントを設置してからエンゲージ

> 情報源: [BoostMatch Ghost Captain 攻略](https://boostmatch.gg/blog/windrose/articles/windrose-ghost-captain-boss-fight-guide) / [BoostMatch 防具セットガイド](https://boostmatch.gg/blog/windrose/articles/windrose-best-armor-sets-builds-guide)

#### ドロップ

- **Soul Eater**（Strength 特化グレートソード、F キーでヘルスドレイン / クールダウン2分）
