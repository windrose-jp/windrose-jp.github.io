# ボス攻略

> 情報源: [Steam ストアページ](https://store.steampowered.com/app/3041230/Windrose/) / [Boostmatch Boss Guide: Thomas Richards](https://boostmatch.gg/blog/windrose/articles/how-to-beat-thomas-richards-windrose-boss-guide) / [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/)

## ボスの概要

各バイオームに固有のボスが存在します。ボス戦はソウルライクスタイルの真価が発揮される場面で、**行動パターンの把握**とスタミナ管理が攻略の鍵です。

ボスを撃破すると **該当バイオームの上位ギア製作・強化枠が解放** されます（例: Coastal Jungle のボス撃破 → Gear Lv6 解放）。

---

## ボス戦の共通原則

### 1. Guard Shield（ポイズバリア）の理解

ほぼすべてのボスは **HPバーの下にGuard Shield（ポイズバリア）を持つ** 。

- Guard Shieldが残っている間は **ダメージが大幅に軽減** される
- **Heavy Attack（重攻撃）** が最も効率的にShieldを削る
- Light Attack（軽攻撃）はほぼ効果がない
- パリィもShield削りに有効
- Shield破壊 → ボスがスタガー → 集中攻撃のチャンス

### 2. スタミナ管理

- **スタミナを青に保つ**: ボス戦でもスタミナ管理が最重要
- Marathon Runner タレントでスタミナ消費を削減済みなら回避の余裕が生まれる
- スタミナ切れは即死リスク

### 3. パターン観察

- 最初の対峙は攻撃せずに **敵の動きを見る**
- 攻撃モーションの予備動作を覚える
- 連続攻撃の回数と後隙を把握する

### 4. 戦前準備

- 食事バフ2枠 + エリクサー1枠を事前に摂取
- 包帯10個以上を常備
- Doctorのポーションを受け取ってから向かう
- ファストトラベルベルで近接地点まで移動
- **焚き火でRestedバフを取得** してから突入

## Thomas Richards（トーマス・リチャーズ）

Coastal Jungle の最初のボス。クエスト「**Revenge is Best Served Cold**」で挑むことになる。

### 攻撃パターン

| パターン | 特徴 | 対処 |
|---------|------|------|
| **2連斬り（固定）** | 必ず2回連続で斬ってくる | 1撃目を回避 → 2撃目をパリィ または 2撃目まで回避してから反撃 |
| 距離詰めダッシュ | ギャップを詰めてくる | サイドステップで側面を取る |

**最大の特徴**: 攻撃が常に **2連斬り固定**。1発目だけ避けて反撃すると2発目を必ず貰うので、**2発分まとめて処理する意識** が必要。

### 推奨ビルド

**最適武器**: **グレートソード（Greatsword）**
- 理由1: リーチが長く、安全な距離から攻撃可能
- 理由2: 重攻撃でGuard Shieldを高効率で削れる
- 理由3: **ブロック時のチップダメージを軽減** できる

**代替武器**:
- ハルバード — リーチと範囲で安定
- ボーンブレイカー — 打撃でポイズ削り最速
- マスケット — 開幕に一発撃ち込むと削りが楽

**非推奨**:
- レイピア — リーチ不足で2連斬りを避けにくい
- 銃単体 — 弾薬不足時に詰む

### 推奨防具
- **Pikeman Set**（両手+15%・HP+200） — グレートソード運用と相性◎
- または **Conquistador Set**（Bulwark） — 攻撃中のよろけを無効化

### 戦術手順

1. **開幕**: マスケットで一発撃ち込み、Shieldを少し削る
2. **接近戦**: 2連斬りの予備動作を見たら **1発目を回避**（横移動）
3. **反撃タイミング**: 2連斬り終了直後の後隙に重攻撃
4. **ラリー活用**: 被弾した時はすぐ攻撃を当ててRallyでHP回復
5. **Shield破壊後**: スタガー中に最大火力コンボを叩き込む
6. **リトリート**: スタミナが半分切ったら一度離れて回復を待つ

### Co-op戦術

- **1名タンク**（Toughguy / Conquistador）が2連攻撃を受け止める
- **他の2〜3名が側面** からポイズを削る
- タンクは **HP半分でスイッチ** の合図を出す
- 復活役（Toughguy）が必ず1人後方にいること

## ボス共通の注意事項

### 死亡ペナルティ

- 装備・回復薬・食料は **保持される**
- 敵HPも回復しないため **リトライ可能**
- 死亡回数に応じたペナルティは情報収集中

### 撃破後の解放要素

- 次のボスが Discovery タブで解放される
- バイオーム固有のギア製作レシピが追加される
- 一部のクエスト進行が進む
