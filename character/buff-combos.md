# バフ最適コンボ集

食事2枠 + エリクサー1枠の3バフを**目的別に最適化**したコンボ集です。レシピの詳細や個別効果は[食事・ポーション](buffs.md)・[錬金術](../crafting/alchemy.md)を参照してください。

> 情報源: [Neonlightsmedia Food & Alchemy](https://www.neonlightsmedia.com/blog/windrose-food-recipes-cooking) / [Neonlightsmedia Potion Recipes](https://www.neonlightsmedia.com/blog/windrose-potion-recipes-alchemy) / [DeltiaGaming](https://deltiasgaming.com/category/windrose/) / [Method.gg Stats](https://www.method.gg/windrose/best-stats-and-talents-to-level-up-first-in-windrose)

---

## バフシステムの基本

| バフ枠 | 種別 | 同時使用数 | 持続時間 |
|--------|------|----------|---------|
| 食事枠1・2 | 料理全般 | **最大2種** | 15〜30分 |
| エリクサー枠 | Elixir / Grog | **1種** | 15〜30分 |
| Rested バフ | Bonfire バフ | 1種 | 最大30分 |

**食事2 + エリクサー1 + Rested = 合計4バフ**が最大構成。遠征前に拠点の Bonfire で Rested を取得してから出発すること。

---

## 目的別 推奨コンボ

### ボス戦（耐久重視）

生存性を最優先。ダメージを受けながら戦う長期戦向け。

| 枠 | アイテム | 効果 |
|----|---------|------|
| 食事1 | **Hard Grog** | DR（被ダメ軽減）+15% / 30分 |
| 食事2 | **Gazpacho** または **Spicy Chicken with Sweet Potato** | AGI+20 / VIT+10 |
| エリクサー | **Elixir of Concentration** | クリティカルダメージ +30% / 15分 |

> **Hard Grog が最優先**。DR+15% は全状況で最も死亡率を下げる。Grog は食事枠を消費するので構成に注意。

---

### ボス戦（火力重視）

短時間で削り切るバースト構成。HP 管理に自信がある場合向け。

| 枠 | アイテム | 効果 |
|----|---------|------|
| 食事1 | **Gazpacho** | AGI+20 / 30分 |
| 食事2 | **Coffee** | END+20（スタミナ燃費改善）/ 30分 |
| エリクサー | **Elixir of Cruelty** | 全ダメージ +8% / 15分 |

> **Mastery をボス前にリスペック**してクリティカル率を上げると、Elixir of Concentration のクリティカルダメージバフとシナジーが高い。

---

### 近接ビルド（通常探索）

近接武器でフィールドを戦いながら移動する標準構成。

| 枠 | アイテム | 効果 |
|----|---------|------|
| 食事1 | **Gazpacho** | AGI+20 / 30分 |
| 食事2 | **Hearty Egg Broth** または **Chowder** | AGI+10 / 30分 |
| エリクサー | **Elixir of Firm Hand** | 近接ダメージ +10% / 15分 |

---

### 遠距離ビルド（銃・マスケット）

遠距離武器メインの構成。

| 枠 | アイテム | 効果 |
|----|---------|------|
| 食事1 | **Banana Muffin** | PRE+10 / 30分 |
| 食事2 | **Coffee** | END+20 / 30分 |
| エリクサー | **Elixir of Precision** | 遠距離ダメージ +13% / 15分 |

> **Gunroom Grog**（リロード速度 +15%）を食事枠に入れてもよい。Elixir of Precision との二重強化で銃火力が大幅に上がる。

---

### 採掘・素材収集

戦闘不要の採掘・伐採作業向け。スタミナと重量効率を最大化。

| 枠 | アイテム | 効果 |
|----|---------|------|
| ステータス | **Endurance を一時的にフル振り**（ステータスリスペック） | スタミナ最大化 |
| 食事1 | **Coffee** | END+20 / 30分 |
| 食事2 | **Coconut Milk with Bananas** または **Bean Soup** | スタミナ増 / AGI+5 |
| エリクサー | 不要（戦闘なし） | — |

> 採掘拠点への往復は **Rested バフ**でスタミナ消費を抑えながら行うと快適。帰還時に Endurance → 戦闘ステータスへリスペックすること。

---

### 探索・ダンジョン

未知のエリアを探索する汎用構成。生存と機動力を両立。

| 枠 | アイテム | 効果 |
|----|---------|------|
| 食事1 | **Spicy Chicken with Sweet Potato** | VIT+10 / 30分 |
| 食事2 | **Coffee** | END+20 / 30分 |
| エリクサー | **Elixir of Cruelty** | 全ダメージ +8% / 15分 |
| 携行品 | **Homeward Journey ポーション** | 緊急テレポート |

> 探索中は**重量を 50% 以下に維持**。100% 超えでドッジコスト2倍になる。定期的に船に素材を戻すルーティンを意識する。

---

### 海戦

船に搭乗しての海上戦闘向け。修理効率と砲撃機会を最大化。

| 枠 | アイテム | 効果 |
|----|---------|------|
| 食事1 | **Hard Grog** | DR+15% / 30分（被弾軽減） |
| 食事2 | **Deadeye Grog** | DR+15% / 30分（同効果・異なるレシピ） |
| ホットバー5〜8 | **Combat Repair Kit** | 10秒で船体 30% 回復（戦闘中使用可） |

> **海戦での食事は Grog 系がほぼ最強**。個人HPよりも船体ダメージが死因になりやすいため、食事よりも船の修理キットと Naval Tactics の選択が勝敗を左右する。

## Elixir（エリクサー）全種一覧

Alchemy Table で製造。食事とは別枠で使用可能。

| エリクサー | 効果 | 持続 | 主素材 |
|-----------|------|------|--------|
| **Elixir of Cruelty** | 全ダメージ +8% | 15分 | Alchemical Base / Boar Tusk / Pearl |
| **Elixir of Precision** | 遠距離ダメージ +13% | 15分 | Alchemical Base / Goat Horn / Wolf Fang |
| **Elixir of Concentration** | クリティカルダメージ +30% | 15分 | Alchemical Base / Bezoar / Aloe ×2 |
| **Elixir of Firm Hand** | 近接ダメージ +10% | 15分 | Alchemical Base 系 |
| **Elixir of Pain Relief** | 被ダメージ軽減 -15% | 15分 | Alchemical Base 系 |

> Alchemical Base は錬金術台で複数の素材から生成。詳細は[錬金術](../crafting/alchemy.md)を参照。

---

## Grog（グロッグ）一覧

Epic 料理（30分持続）。食事枠を消費することに注意。

| グロッグ名 | 効果 | 主素材 |
|-----------|------|--------|
| **Hard Grog** | 被ダメージ軽減 DR +15% | Rum ×4 / Brandy ×2 / Lime ×3 |
| **Gunroom Grog** | 銃リロード速度 +15% | Rum ×4 / Whiskey ×2 / Cayenne ×3 |
| **Deadeye Grog** | 被ダメージ軽減 DR +15% | Rum ×4 / Madeira ×2 / Coffee Beans ×1 |

> Hard Grog と Deadeye Grog は効果が同じ DR+15%。素材が入手しやすい方を使用すること。

## Homeward Journey（帰還ポーション）

探索時の保険として携行推奨。

| 項目 | 内容 |
|------|------|
| 効果 | 直近のリスポーン地点へ**即時テレポート** |
| 必要 | Alchemy Table Lv2（Stove and Pot 追加） |
| 主素材 | Alchemical Base ×3 / Rum Bottle ×1 / Feather ×5 / Undead Essence ×1 |
| 解放条件 | 各素材を1度以上入手（Discovery 経由） |
| エリクサー枠？ | **食事枠とは独立した枠で使用可能**（詳細要確認）|

深いダンジョンや遠い島を探索する際に1本携行しておくと帰還コストがゼロになる。
