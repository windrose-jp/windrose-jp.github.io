# 食事・ポーション

> 情報源: [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/) / [Boostmatch 料理ガイド](https://boostmatch.gg/blog/windrose/articles/windrose-complete-food-guide-all-items-buffs-recipes) / [allthings.how HP回復ガイド](https://allthings.how/every-way-to-restore-health-in-windrose/) / [MetaForge ビギナーズガイド](https://metaforge.app/news/beginners-guide-windrose)

## バフシステムの概要

Windroseでは**食事**と**ポーション/エリクサー**を使って一時的なバフ効果を得られます。

- **食事バフは同時に2枠**まで有効
- **3枠目を食べると最古のバフが上書き**される
- **ポーション系バフは食事バフと別枠**で同時スタック可能
- 遠征前に拠点で両方補充してから出発する
- 料理台（Cooking Fire）はマップのどこでも設置できるので遠征先でも調理可能

### 食事バフの持続時間

| 食品タイプ | 持続時間 |
|----------|--------|
| **生食（Raw）** | **7 分**、ステータス上昇なし |
| **Lv1 ステーション調理 / Common・Uncommon 料理** | **15 分** |
| **Rare・Epic 料理** | **30 分** |

**ボス戦前は Rare/Epic で30分バフを2枠埋める**のがセオリー。

## HP 回復手段の優先度

| 優先度 | 方法 | 説明 |
|--------|------|------|
| 1 | **ラリー（Rally）** | 被弾直後にすぐ攻撃 → HPの一部を回復。消耗ゼロ |
| 2 | **焚き火（Bonfire）** | 近くにいると受動回復（**ダメージを受けても止まらない**） |
| 3 | **包帯（Bandage）** | 30秒かけて900HP回復（ダメージで中断される） |
| 4 | **ポーション** | ドクターから毎日1本無料。緊急時用 |

## 回復アイテム

### 包帯（Bandage）

| 項目 | 内容 |
|------|------|
| クラフト | Workbench で Coarse Fiber ×1 |
| 効果 | **30秒かけて900HP回復** |
| 注意点 | 回復中にダメージを受けるとキャンセルされる |
| 推奨 | 常時10本以上携行 |

### ヘルスポーション（Health Potion）

| 種別 | 回復量 | 備考 |
|------|-------|------|
| **Minor Healing Potion** | **最大HPの 35%**（即時） | 序盤〜中盤 |
| **Standard Healing Potion**（Healing Potion） | **最大HPの 55%**（即時） | 中盤〜 |

| 項目 | 内容 |
|------|------|
| クラフト | Alchemy Table で Clay + 植物素材 |
| 無料入手 | **Doctor Galen に Healing Herbs を渡すと薬を調合** |
| 特徴 | 即時回復。**ボス戦では Bandage より Potion 推奨**（Bandageは被弾で切れる） |

> ⚠️ **Bonfire を撤去すると Galen が消える**。再建立で再召喚可能だが、拠点移動時は注意。

## エリクサー（Elixir）

戦闘能力を一時的に強化するポーション。Alchemy Table で製造。**食事バフと別枠でスタック可能**なため、ボス戦前に必ず1本飲んでおく。

| エリクサー名 | 効果 | 備考 |
|------------|------|------|
| **Elixir of Concentration** | **クリティカルダメージ +30%** | Crit ビルド必須（Reliable Musket Marksman 等） |
| **Elixir of Cruelty** | **全般ダメージ +8%** | 汎用最強。武器種問わず |
| **Elixir of Firm Hand** | **近接ダメージ +13%** | 近接ビルド向け |
| **Elixir of Pain Relief** | 被ダメージ軽減 | タンク系・ボス連戦用 |
| **Elixir of Precision** | 遠距離ダメージ増加 | Marksman 系 |

> 情報源: [allthings.how Best Builds](https://allthings.how/windrose-best-builds-three-loadouts-that-dominate-combat/)

Alchemy Table の解放: **Misty Orchid（霧の蘭）** を素材として使用（Foothills で入手）。

## Rested バフ（コンフォートシステム）

> 情報源: [Boostmatch Comfort Guide](https://boostmatch.gg/blog/windrose/articles/windrose-comfort-guide-decorations-rested-buff-stamina)

拠点の Bonfire 近くにいると **Rested バフ**が発動し、スタミナ回復速度と最大量が大幅に上昇する。

| 状態 | 持続時間 |
|------|--------|
| 裸の Bonfire | **数分程度** |
| デコレーション配置で Comfort Level 最大 | **最大 30 分** |

### Comfort Level の上げ方

Bonfire 近くに**サブカテゴリごとに1個ずつ**デコレーションを配置すると Comfort Level が上がる：

| サブカテゴリ | 例 |
|------------|-----|
| Seating（椅子） | 椅子・ベンチ |
| Table（机） | テーブル |
| Bed（ベッド） | Simple Bed 等 |
| Lighting（照明） | 追加の灯り |
| Storage（収納） | チェスト |
| Trophy（トロフィー） | Animal Head Trophy |

**同カテゴリを複数置いても +1 のみ**。各カテゴリから1個ずつ配置することが重要。

### Animal Head Trophy の特殊仕様

- **固有種ごとに +1 の Comfort**
- **最大 12 個まで**累積可能
- 狩猟で多様な動物を倒すことで大幅に強化できる

### Comfort Station 建造物

- **建造物「Comfort Station」配置で Rested 持続時間 +30%**
- **スタック可能**（複数置くとさらに延長）

### 運用方針

- 遠征前に必ず Rested を取得してから出発
- スタミナ燃費が別物になるため、戦闘・採集両方で恩恵大
- 拠点レイアウトは Bonfire 周辺に多カテゴリのデコレーションを集約

## ラリーシステム（Rally）

**被弾直後にすぐ攻撃を当てると、受けたダメージの一部を回復できる。**

- HP 消費ゼロの回復手段なので最優先で活用する
- Toughguy タレント「**You Will Answer for This**」で効果を強化できる
- 出血（Bleed）ビルドとの相性が良い
