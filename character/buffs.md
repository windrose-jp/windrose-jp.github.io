# 食事・ポーション

> 情報源: [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/) / [Boostmatch 料理ガイド](https://boostmatch.gg/blog/windrose/articles/windrose-complete-food-guide-all-items-buffs-recipes) / [allthings.how HP回復ガイド](https://allthings.how/every-way-to-restore-health-in-windrose/) / [MetaForge ビギナーズガイド](https://metaforge.app/news/beginners-guide-windrose)

## バフシステムの概要

Windroseでは**食事**と**ポーション/エリクサー**を使って一時的なバフ効果を得られます。

- **食事バフは同時に2枠**まで有効
- 遠征前に拠点で両方補充してから出発する
- 料理台（Cooking Fire）はマップのどこでも設置できるので遠征先でも調理可能

---

## HP 回復手段の優先度

| 優先度 | 方法 | 説明 |
|--------|------|------|
| 1 | **ラリー（Rally）** | 被弾直後にすぐ攻撃 → HPの一部を回復。消耗ゼロ |
| 2 | **焚き火（Bonfire）** | 近くにいると受動回復（**ダメージを受けても止まらない**） |
| 3 | **包帯（Bandage）** | 30秒かけて900HP回復（ダメージで中断される） |
| 4 | **ポーション** | ドクターから毎日1本無料。緊急時用 |

## 回復アイテム

### 包帯（Bandage）

| 項目 | 内容 |
|------|------|
| クラフト | Workbench で Coarse Fiber ×1 |
| 効果 | **30秒かけて900HP回復** |
| 注意点 | 回復中にダメージを受けるとキャンセルされる |
| 推奨 | 常時10本以上携行 |

### ヘルスポーション（Health Potion）

| 項目 | 内容 |
|------|------|
| クラフト | Alchemy Table で Clay + 植物素材 |
| 無料入手 | **Doctor Galen から毎日1本無料**（忘れずに受け取ること） |
| 特徴 | 即時回復。包帯より高速だがコストが高い |

## エリクサー（Elixir）

戦闘能力を一時的に強化するポーション。Alchemy Table で製造。

| エリクサー名 | 効果 |
|------------|------|
| **Elixir of Concentration** | クリティカルダメージ強化 |
| **Elixir of Cruelty** | 全般的なダメージ増加 |
| **Elixir of Firm Hand** | 近接ダメージ増加 |
| **Elixir of Pain Relief** | 被ダメージ軽減 |
| **Elixir of Precision** | 遠距離ダメージ増加 |

Alchemy Table の解放: **Misty Orchid（霧の蘭）** を素材として使用（Foothills で入手）。

---

## Rested バフ（コンフォートシステム）

拠点の Bonfire 近くにいると **Rested バフ**が発動し、スタミナ回復速度が大幅に上昇する。

| 要素 | 効果 |
|------|------|
| Bonfire 近くに滞在 | Rested バフ発動 |
| **拠点のデコレーション数を増やす** | **持続時間が延長（最大30分）** |
| Comfort レベル向上 | スタミナバー上限増加・回復速度向上 |

遠征前に Rested バフを満タンにしてから出発するのが基本。

---

## ラリーシステム（Rally）

**被弾直後にすぐ攻撃を当てると、受けたダメージの一部を回復できる。**

- HP 消費ゼロの回復手段なので最優先で活用する
- Toughguy タレント「**You Will Answer for This**」で効果を強化できる
- 出血（Bleed）ビルドとの相性が良い
