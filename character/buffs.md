# 食事・ポーション

> 情報源: [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/) / [Boostmatch Food Guide](https://boostmatch.gg/blog/windrose/articles/windrose-complete-food-guide-all-items-buffs-recipes) / [MetaForge ビギナーズガイド](https://metaforge.app/news/beginners-guide-windrose) / [All Things How: HP回復方法](https://allthings.how/every-way-to-restore-health-in-windrose/)

## バフシステムの概要

Windroseでは**食事**と**ポーション（エリクサー）**を使って一時的なバフ効果を得られます。

- **同時に2つの食事バフ**を装備可能（飲料込みで3枠の場合あり）
- **エリクサーは食事枠と別枠** で付与されるため、食事2 + エリクサー1 の同時運用が可能
- 序盤から積極的に活用することで生存性が大きく向上する
- 遠征前に拠点で補充してから出発する習慣をつけると良い

## HP回復手段一覧

### 1. ラリー（Rally）— 最速回復

**被弾直後にすぐ攻撃を当てると、受けたダメージの一部を回復できる。**

- 被弾の「赤ゲージ」が残っている間に攻撃を当てることで発動
- タレントで強化可能
- 出血（Bleed）ビルドとの相性が良く、攻撃を続けながらHPを維持できる
- **ボス戦で最も重要な回復手段**

### 2. 焚き火（Bonfire）による自然回復

拠点やフィールドに設置した焚き火の近くにいると**パッシブ回復**が発動する。

- **ダメージを受けても回復がキャンセルされない** 点が包帯との大きな違い
- 戦闘中の立ち回りポジションとして活用可能
- **Restedバフ**（後述）の発生源でもある

### 3. 包帯（Bandage）

| 項目 | 内容 |
|------|------|
| クラフト | 作業台（Workbench）で粗い繊維（Coarse Fiber）× 1 |
| 効果 | **30秒かけて900HP回復** |
| 注意点 | 回復中にダメージを受けるとキャンセルされる |

序盤の主力回復手段。植物繊維から作れる粗い繊維でコストが低く、**序盤は10個以上常備** が推奨される。

### 4. ヘルスポーション（Health Potion）

| 項目 | 内容 |
|------|------|
| クラフト | 錬金テーブル（Alchemy Table）で粘土＋発光草＋赤ヒール草 |
| 入手方法 | クラフト / 宝箱からドロップ / **Doctor（ドクター）NPCから毎日1本無料** |

**Doctor（ドクター）** は拠点にいるNPCで、毎日ポーションを1本無料でくれる。**忘れずに受け取ること**。

## Restedバフ（コンフォートシステム）

**焚き火（Bonfire）の近くで発生する最重要バフ**。

- スタミナ回復速度が大幅に上昇
- スタミナ最大量も増加
- 拠点のデコレーション（Comfort値）を増やすと **最大30分** まで持続時間が延びる
- 戦闘継続力がまるで別物になるため、遠征前に必ず取得すべき

**Comfort値を上げる方法**:
- 家具・装飾品を設置する
- ベッドを設置する
- 拠点の屋根・壁を整える

## エリクサー（Elixir）一覧

食事枠とは別に付与される強化薬。**ボス戦の前に飲むのが定石**。錬金テーブル（Alchemy Table）でクラフト。

| エリクサー | 効果 |
|----------|------|
| Elixir of Concentration（集中のエリクサー） | クリティカルダメージ強化 |
| Elixir of Cruelty（残虐のエリクサー） | ダメージ全般強化 |
| Elixir of Firm Hand（確固たる手のエリクサー） | 近接ダメージ強化 |
| Elixir of Pain Relief（苦痛緩和のエリクサー） | 被ダメージ軽減 |
| Elixir of Precision（精密のエリクサー） | 遠距離ダメージ強化 |

## ヤシの実（Coconut）— 序盤の万能バフ

| 項目 | 内容 |
|------|------|
| 入手 | ヤシの木を**揺らす**（切らない） |
| 効果 | **最大HPが7分間増加** |
| 備考 | 切り倒すと復活が遅いので伐採は避ける |

序盤は回復手段が限られるため、ヤシの実の **最大HP増加バフ** が実質的な回復量底上げになる。

---

## 運用のベストプラクティス

1. **遠征前にバフ補充**: 食事2枠＋エリクサー1枠を拠点で用意
2. **Restedバフ取得**: 出発直前に焚き火で発生させる
3. **Doctorから毎日ポーション受領**: ログインごとに確認
4. **包帯は10個以上常備**: 道中のクラフト施設で追加作成可能
5. **ボス戦前は食事入れ替え**: 道中はEndurance系、ボス戦前はAttack系に切り替え
