# 勢力・評判

> 情報源: [Method.gg Factions and Reputations](https://www.method.gg/windrose/windrose-factions-reputations-all-faction-items-best-faction-order) / [Boostmatch Brethren Reputation](https://boostmatch.gg/blog/windrose/articles/windrose-brethren-of-the-coast-reputation-guide) / [Boostmatch Faction Guide](https://boostmatch.gg/blog/windrose/articles/windrose-faction-guide-reputation-system) / [Gamersandgeek](https://gamersandgeek.com/how-factions-work-in-windrose/)

## ファクション（勢力）システム

Windrose には複数のファクションが登場し、それぞれと **評判（Reputation）** を構築できる。評判レベルに応じて**設計図・船バリエーション・装備**が解放される。

## 確認されている勢力

| 勢力名 | 概要 |
|--------|------|
| **Brethren of the Coast（海岸同盟）** | 海賊同盟の主要勢力。Brigantine / Frigate 設計図解放の主軸 |
| **Blackbeard（黒ひげ派閥）** | 黒ひげ側。Blackbeard バリエ船の解放元（要検証） |
| **People of Tortuga（トルトゥガの民）** | トルトゥガ都市の住民・商人 |
| **Rogue Buccaneers（ローグ・バッカニア）** | 独立系の荒くれ海賊。Glorious Hunters などのクエストに関与。Buccaneer's Friend（最強遠距離武器）解放条件の派閥 |
| **Smugglers of Port Royal（ポートロイヤルの密輸業者）** | 地下の裏市場派閥。**他商人が買わない密輸品・贅沢品・古代遺物を買取**。Underground Network クエストで解放 |

## 評判の上げ方

以下の複数経路で派閥レピュテーションが上昇する：

| 方法 | 内容 |
|------|------|
| **Reputation Item の引換** | 特定アイテムを派閥NPCに渡す |
| **クエスト報酬** | 派閥に紐づくクエストを完了 |
| **敵艦撃沈** | **Insignia（紋章）取得**で該当派閥への評判上昇 |

## Brethren of the Coast 評判の解放報酬（重要）

| レピュ Lv | 解放内容 |
|---------|---------|
| **Lv 2** | **Brigantine（中型戦闘艦）設計図** → Tortuga で購入可能 |
| **Lv 4** | **Frigate（大型戦闘艦）設計図** → Tortuga で購入可能 |

**Ketch → Brigantine → Frigate** と船の大型化ルートは Brethren 評判で解放される。Brethren クエを積極的にこなすのが中盤の進行鍵。

## Smugglers of Port Royal（密輸業者）— 事実上のブラックマーケット

> 情報源: [gamerblurb Smugglers](https://gamerblurb.com/articles/windrose-where-to-find-smugglers) / [FextraLife Smugglers](https://windrose.wiki.fextralife.com/Smugglers) / [Boostmatch Tortuga Guide](https://boostmatch.gg/blog/windrose/articles/windrose-tortuga-guide) / [NeonLightsMedia Traders](https://www.neonlightsmedia.com/blog/windrose-faction-traders-locations)

Tortuga の通常商人では売却できない**密輸品・贅沢品・古代遺物**を買い取ってくれる裏の派閥。冒険で溜まる「売れないガラクタ」を換金できる重要窓口。

### 場所

**Tortuga 南東の洞窟** "Smugglers Waters" の内部。**Marita** というNPCの隣に buyer が常駐する。

### アンロック手順（Underground Network クエスト）

| 手順 | 内容 |
|------|------|
| 1 | 密輸業者のキャンプで **"Letter to a Good Friend"** を拾う |
| 2 | クエスト **Underground Network** が発生 |
| 3 | 目標 "Talk to Marita" を進める |
| 4 | 洞窟内部で Marita と会話 → 隣の buyer が解放 |
| 5 | マップに密輸業者の位置が表示される |

### 取扱品（買取専門）

| 品目 | 価格 |
|------|------|
| **Contraband（密輸品）** | 150 Piastre |
| **Luxuries（贅沢品）** | 250 Piastre |
| Insignia / 銀食器 / ビーズネックレス / ボーンビーズ 等 | 5〜10 Piastre |
| **Ancient Items（古代遺物: Gold Mask of the Priest, Ancient Incense Burner 等）** | **1〜5 Guinea**（高級通貨） |
| Provisions / Medicine | — |

> アイテム説明に **"can be sold to smugglers"** と書いてある品は**ここでしか捌けない**。アンロック前のプレイヤーはこれらが倉庫に死蔵されがちなので、序盤〜中盤のうちに Underground Network を進めておくのが得。

### 注意: 「Smuggler's Cache」チェストとは別物

スタート地点近くの隠しチェスト **Smuggler's Cache**（最初の Fast Travel Bell + レシピ入手場所）は命名フレーバーのみで、この Smugglers 派閥 NPC とは別システム。

## 船の Faction バリエーション

船種には **Stock / Brethren / Blackbeard** の3バリエーションがあり、該当派閥の評判があると購入可能になる：

| バリエーション | 性格 |
|--------------|------|
| Stock | バランス型 |
| **Brethren** | HP +30%、速度 -2kn（耐久重視） |
| **Blackbeard** | HP -30%、速度 +2kn、上位砲装備可（機動・火力重視） |

詳細は [船の種類](../ships/ship-types.md) を参照。

## Rare 装備設計図の入手経路

中盤装備の多くは派閥系で解放される：

| 入手経路 | 内容 |
|---------|------|
| **Pirate Camp 襲撃** | Rare 装備設計図のドロップ |
| **Tortuga Faction Provisioner** | 評判レベルに応じて購入可能な設計図 |

Foothills 突入前に Rare セットを揃えたい場合は、派閥クエ+Pirate Camp 掃除が効率的。

## 派閥間の関係

- 一方の勢力と友好的になると他勢力との関係に影響する可能性あり（詳細仕様は要検証）
- 序盤は **Brethren of the Coast** を優先するのが船の大型化ルート的に効率的
