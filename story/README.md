# ストーリー概要

> 情報源: [Steam ストアページ](https://store.steampowered.com/app/3041230/Windrose/)

## 世界観

Windroseは**海賊時代の代替歴史（Alternative History）**を舞台にしています。現実の歴史上の人物（黒ひげ / Blackbeard）が登場し、超自然的な力が絡み合う独特の世界観です。

## ストーリーの概要

- プレイヤーは船長として復讐と生存の物語を歩む
- **黒ひげ（Blackbeard）**に挑戦することが物語の核心
- 帝国・海賊団・謎の暗黒勢力が絡む三つ巴の争いが展開
- メインストーリーの完了まで**約50〜70時間**（プレイスタイルによる）
