# マルチプレイ

> 情報源: [Steam ストアページ](https://store.steampowered.com/app/3041230/Windrose/) / [Steam ニュースRSSフィード](https://store.steampowered.com/feeds/news/app/3041230/)

## マルチプレイの概要

Windroseは**オンラインCo-op（協力プレイ）**に対応しています。現在は最大4人でのプレイが可能で、ソロよりも難しい場面を仲間と協力して乗り越えられます。

## Co-opの参加方法

| 方式 | 説明 |
|------|------|
| ホスト型セッション | プレイヤーが自分のゲームをホストして友人を招待 |
| 専用サーバー | **Early Access開始と同時にサポート開始**（Nitrado等を利用） |
| ダイレクトIP接続 | サーバーIPを直接入力して接続（Patch 0.10.0.3.104で追加） |

### 招待コードについて

友人を招待する際は**招待コード（Invite Code）**を使用します。  
コードの文字数は **1〜32文字** で自由に設定できます（2026年4月17日のHotfixで制限が緩和されました）。

## Co-opの注意点

- ホスト型セッションではホストがゲームを終了するとセッションが閉じる
- 専用サーバーを使用すればホスト不在でも稼働継続が可能
- **レピュテーション（評判）はCo-opで共有されるかどうか**はプレイヤー間で議論中（Steam Discussionsより）

## ソロプレイとの違い

ソロでも全コンテンツにアクセス可能です。Co-opでは複数人で役割分担ができるため、難しいボス戦や海戦が楽になります。船では**舵取り・砲撃手・甲板員**などの役割分担が自然に生まれます。
