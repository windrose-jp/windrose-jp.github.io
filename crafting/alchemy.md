## Alchemy Table（錬金テーブル）の解放

**第2島で Misty Orchid（霧の蘭）を入手**すると Alchemy Table のレシピが解放される。

| 手順 | 内容 |
|------|------|
| 1 | 第2島（Foothills）に到着 |
| 2 | Misty Orchid を発見・入手 |
| 3 | Alchemy Table のクラフトレシピが解放（Discovery タブで確認） |
| 4 | 屋根付きの建物内に Alchemy Table を設置（Bonfire 範囲内推奨） |

---

## Alchemy Table のアップグレード

### Lv2：Stove and Pot の追加

Alchemy Table に **Stove and Pot（かまどと鍋）** を隣接設置すると Lv2 になり、上位レシピが解放される。

| アドオン | 素材 |
|---------|------|
| Stove and Pot | Stone ×5 / Clay ×5 / Copper Pot ×1 |

> **Homeward Journey ポーション**や上位 Elixir は Lv2 以上が必要。

## ポーション一覧

### Minor Healing Potion（小回復ポーション）

| 項目 | 内容 |
|------|------|
| 効果 | 最大HPの **35%** を即時回復 |
| 主素材 | Alchemical Base / Healing Herbs |
| 入手 | Doctor Galen から毎時1本無料 |

### Standard Healing Potion（回復ポーション）

| 項目 | 内容 |
|------|------|
| 効果 | 最大HPの **55%** を即時回復 |
| 主素材 | Alchemical Base / 上位の回復素材 |

### Homeward Journey（帰還ポーション）

直近のリスポーン地点（Bonfire / Tent）へ**即時テレポート**する緊急脱出用ポーション。深いダンジョンや遠方の島を探索するときに1本携行しておくと帰還コストがゼロになる。

| 項目 | 内容 |
|------|------|
| 必要施設 | **Alchemy Table Lv2**（Stove and Pot 追加後） |
| 主素材 | Alchemical Base ×3 / Rum Bottle ×1 / Feather ×5 / Undead Essence ×1 |
| 解放条件 | 各素材を1度以上入手（Discovery 経由で自動解放） |

## Elixir（エリクサー）一覧

食事バフとは**別枠**で使用できる戦闘強化ポーション。持続時間は基本 15 分。

| エリクサー名 | 効果（数値） | 主素材 |
|------------|------------|--------|
| **Elixir of Cruelty** | 全ダメージ **+8%** | Alchemical Base / Boar Tusk / Pearl |
| **Elixir of Firm Hand** | 近接ダメージ **+10%** | Alchemical Base 系 |
| **Elixir of Precision** | 遠距離ダメージ **+13%** | Alchemical Base / Goat Horn / Wolf Fang |
| **Elixir of Concentration** | クリティカルダメージ **+30%** | Alchemical Base / Bezoar / Aloe ×2 |
| **Elixir of Pain Relief** | 被ダメージ軽減 **-15%** | Alchemical Base 系 |

> 素材の詳細レシピは一部未確認【要検証】。素材を入手すると Discovery タブ経由でレシピが開放される。

## Grog（グロッグ）一覧

食事枠を消費する Epic 品質の飲料バフ。持続時間 30 分で Rare/Epic 料理と同等の扱い。

| グロッグ名 | 効果 | レシピ |
|-----------|------|--------|
| **Hard Grog** | 被ダメージ軽減 DR **+15%** | Rum ×4 / Brandy ×2 / Lime ×3 |
| **Gunroom Grog** | 銃リロード速度 **+15%** | Rum ×4 / Whiskey ×2 / Cayenne ×3 |
| **Deadeye Grog** | 被ダメージ軽減 DR **+15%** | Rum ×4 / Madeira ×2 / Coffee Beans ×1 |

> **注意**: Grog は食事枠を消費する。エリクサーとは異なる枠扱いの確認が取れていない場合は[バフ最適コンボ集](../character/buff-combos.md)も参照。

## Alchemical Base（錬金ベース）の入手

Elixir 系全般に使う素材。以下の方法で入手できる。

- Alchemy Table で植物素材を組み合わせて生成
- フィールドの隠しチェストや NPC からの購入
- 詳細レシピは**素材を1個拾うと Discovery 経由で解放される**

---

## 素材の主な入手先

| 素材 | 主な入手先 |
|------|-----------|
| Healing Herbs（回復ハーブ） | 第2島（Foothills）に15個前後のまとまり場所あり |
| Boar Tusk（イノシシの牙） | Boar 狩り |
| Animal Fat（動物脂） | Boar / 動物狩り（Lamp の補充にも使用） |
| Pearl（真珠） | Scallop Shell を右クリックで取り出す |
| Feather（羽根） | 鳥系エネミーのドロップ / フィールド採取 |
| Undead Essence | Undead 系エネミーのドロップ |
| Goat Horn / Wolf Fang | 各動物ドロップ |
| Bezoar / Aloe | フィールド採取 |
| Rum Bottle | Trader / 海賊キャンプのチェスト |
