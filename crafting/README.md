# クラフト概要

> 情報源: [Steam ストアページ](https://store.steampowered.com/app/3041230/Windrose/) / [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/)

## クラフトシステムの概要

素材を収集し、クラフトステーションを使って武器・防具・消耗品・建材を作成します。序盤から継続的にステーションを解放・アップグレードしていくことが成長の鍵です。

## クラフトの基本的な流れ

1. 素材（木材・石・鉱石など）を収集
2. 炭焼き窯（Charcoal Kiln）で木炭を製造
3. 製錬炉（Smelting Furnace）で鉱石をインゴットに精錬
4. 各専用ステーションでアイテムをクラフト
