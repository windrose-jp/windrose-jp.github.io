## エンチャントとは

**Enchanting Table（エンチャントテーブル）**で装備に特殊な魔法特性を付与するシステム。

**Ascend（昇格）とは独立した別システム**であることに注意：

| システム | 目的 | 素材 | 施設 |
|---------|------|------|------|
| **Ascend** | Rare → Epic に装備ランク昇格＋固有パッシブ付与 | Tumbaga Ingot | Weaponsmith Workshop |
| **Enchanting** | 装備に魔法特性を追加 | Arborum Ingot | Enchanting Table |

どちらも終盤の装備強化軸だが、Enchanting の方がより上位ティアに位置する。

---

## Enchanting Table の解放

**Swamp（呪われた沼地）または Foothills（山麓）の Ruins（廃墟）で「Ancient Scraps（古代のスクラップ）」を発見すると、Enchanting Table のレシピが自動解放される。**

Ancient Scraps は廃墟内のチェストやフィールドで見つかる。発見後は Discovery タブ（G キー）で確認すること。

---

## Enchanting Table の建築

### 前提施設: Large Smelting Furnace

Enchanting Table の素材（Hewn Stone / Mire Metal Ingot）を生成するために必要。

| 施設 | 素材 |
|------|------|
| **Large Smelting Furnace** | Hewn Stone ×20 / Foothills Iron Ingot ×10 / Clay ×20 |

Large Smelting Furnace は**屋外設置専用**。Bonfire の暖気範囲外でも動作する。

### Enchanting Table のクラフトコスト

| 素材 | 数量 |
|------|------|
| **Mire Metal Ingot** | ×3 |
| **Hewn Stone** | ×10 |
| **Plant Fiber** | ×10 |
| **Essence Arborum** | ×2 |

### 配置要件

| 条件 | 内容 |
|------|------|
| 屋根 | **必須**（Sheltered 判定が必要） |
| Bonfire | **暖気範囲内に設置必須**（範囲外は動作しない） |
| 耐久 | HP 3000（消耗するため修理が必要） |

## Arborum 素材の入手

Enchanting に使う素材は専用の生産ラインが必要。

### Essence Arborum（エッセンス・アーボラム）

**Plague Wood（疫病の木）を Large Smelting Furnace で精錬**して生成。

Plague Wood は Swamp バイオームで伐採できる特殊な木材。

### Tainted Bile（汚染された胆汁）

Swamp の敵（感染した動物など）からドロップ。Enchanting 素材として消費される。

### Ingot Arborum（アーボラムインゴット）

Enchanting Table で以下の素材から生成：

| 素材 | 概要 |
|------|------|
| Mire Metal Ingot | Large Smelting Furnace で生成 |
| Essence Arborum | Plague Wood を精錬 |
| Tainted Bile | Swamp 敵からドロップ |

**Ingot Arborum を使って最高ティアの「Arborum 装備」をクラフト・強化できる**。

## 進行の目安

Enchanting は**Swamp / Foothills 到達後の中〜終盤コンテンツ**。

進行の流れ：
1. Swamp または Foothills の Ruins で Ancient Scraps を入手
2. Large Smelting Furnace を建築（屋外に設置）
3. Plague Wood を伐採して Essence Arborum を精錬
4. Enchanting Table をクラフト・屋根下 Bonfire 範囲内に設置
5. Swamp 敵から Tainted Bile を収集
6. Ingot Arborum を生成して Arborum 装備ラインへ
