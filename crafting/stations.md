# クラフトステーション

> 情報源: [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/)

## クラフトステーション一覧

### 炭焼き窯（Charcoal Kiln）
木材から木炭（Charcoal）を製造する施設。多くのレシピで木炭が必要になるため、序盤に優先して建設します。

| 項目 | 内容 |
|------|------|
| 主な用途 | 木炭の製造 |
| 解放条件 | 情報収集中 |
| 建設素材 | 情報収集中 |

---

### 製錬炉（Smelting Furnace）
鉱石をインゴットに精錬する施設。銅鉱石→銅インゴット、タンバガインゴットなどの精錬に使用。

| 項目 | 内容 |
|------|------|
| 主な用途 | 鉱石の精錬 |
| 精錬できる素材 | 銅（Copper）、タンバガ（Tumbaga）など |
| 解放条件 | 情報収集中 |
| 建設素材 | 情報収集中 |

---

### 錬金テーブル（Alchemy Table）
ポーション・薬品を調合する施設。「Misty Orchid（霧の蘭）」を使って解放します。

| 項目 | 内容 |
|------|------|
| 主な用途 | ポーション・薬品の製造 |
| 解放素材 | ミスティオーキッド（Misty Orchid） |
| 建設素材 | 情報収集中 |

→ 詳細は[錬金術](alchemy.md)ページを参照

---

### 武器鍛冶台（Weaponsmith Station）
武器のアップグレード・強化を行う施設。タンバガインゴット（Tumbaga Ingot）が主要素材として使用されます。

| 項目 | 内容 |
|------|------|
| 主な用途 | 武器のアップグレード・作成 |
| 主要素材 | タンバガインゴット（Tumbaga Ingot） |
| 解放条件 | 情報収集中 |
| 建設素材 | 情報収集中 |

---

---

### 作業台（Workbench）

序盤に最初に建てるべき施設。道具・武器・包帯など基本的なアイテムをほぼすべてここでクラフトする。

| 項目 | 内容 |
|------|------|
| 主な用途 | 道具・武器・包帯・袋などのクラフト |
| クラフトできる主要品 | 石のつるはし、石の斧、包帯、裂けた帆布の袋（持ち物拡張）、たいまつ |
| 解放条件 | 序盤から建設可能 |

クラフトステーションはレベル1・2・3とアタッチメントでアップグレードでき、上位レシピが解放される。

---

### アップグレードステーション（Upgrading Station）

武器・防具・バッグのアップグレードを行う施設。

| 項目 | 内容 |
|------|------|
| 主な用途 | 武器・防具・袋の強化 |
| 解放条件 | 情報収集中 |

---

### 解体台（Dismantling Bench）

不要なアイテムを素材に戻す施設。**素材は100%返却される**が、アップグレードに使った資源は戻らない。

| 項目 | 内容 |
|------|------|
| 主な用途 | 不要品の素材化 |
| 注意点 | アップグレード分の素材は回収不可 |

---

### 料理台（Cooking Fire）

食事バフアイテムをクラフトする施設。**マップのどこにでも設置可能**（拠点外でも使える）。

| 項目 | 内容 |
|------|------|
| 主な用途 | 食事バフアイテムの調理 |
| 特徴 | 拠点外・フィールド上にも設置可能 |

→ 詳細は[料理](cooking.md)ページを参照
