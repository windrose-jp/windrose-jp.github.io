# クラフトステーション

> 情報源: [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/)

## クラフトステーション一覧

### 炭焼き窯（Charcoal Kiln）
木材から木炭（Charcoal）を製造する施設。多くのレシピで木炭が必要になるため、序盤に優先して建設します。

| 項目 | 内容 |
|------|------|
| 主な用途 | 木炭の製造 |
| 解放条件 | 情報収集中 |
| 建設素材 | 情報収集中 |

---

### 製錬炉（Smelting Furnace）
鉱石をインゴットに精錬する施設。銅鉱石→銅インゴット、タンバガインゴットなどの精錬に使用。

| 項目 | 内容 |
|------|------|
| 主な用途 | 鉱石の精錬 |
| 精錬できる素材 | 銅（Copper）、タンバガ（Tumbaga）など |
| 解放条件 | 情報収集中 |
| 建設素材 | 情報収集中 |

---

### 錬金テーブル（Alchemy Table）
ポーション・薬品を調合する施設。「Misty Orchid（霧の蘭）」を使って解放します。

| 項目 | 内容 |
|------|------|
| 主な用途 | ポーション・薬品の製造 |
| 解放素材 | ミスティオーキッド（Misty Orchid） |
| 建設素材 | 情報収集中 |

→ 詳細は[錬金術](alchemy.md)ページを参照

---

### 武器鍛冶台（Weaponsmith Station）
武器のアップグレード・強化を行う施設。タンバガインゴット（Tumbaga Ingot）が主要素材として使用されます。

| 項目 | 内容 |
|------|------|
| 主な用途 | 武器のアップグレード・作成 |
| 主要素材 | タンバガインゴット（Tumbaga Ingot） |
| 解放条件 | 情報収集中 |
| 建設素材 | 情報収集中 |

---

その他のクラフトステーション・調理台・作業台などは情報収集中です。
