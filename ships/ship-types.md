# 船の種類

> 情報源: [Steam ストアページ](https://store.steampowered.com/app/3041230/Windrose/) / [Windrosewiki.org Ship Building Guide](https://windrosewiki.org/blog/windrose-ship-building-guide-101) / [Boostmatch Ship Building Guide](https://boostmatch.gg/blog/windrose/articles/windrose-ship-building-guide-fix-frigate-cutter-alpha)

## 船の区分

Windroseの船は大きく **スタータ艇** と **戦闘艦** の2区分があります。

### スタータ艇

- **Kキーで召喚可能**、どこからでも呼び寄せられる
- **喪失しない**（沈没しても再召喚可）
- 無料で使えるが、耐久・火力は低い
- 序盤の探索・島移動に使う

### 戦闘艦（Combat Ships）

- **Wharf（波止場）** で建造・修理する
- **破壊されうる**（素材を失う）
- 耐久・火力・積載量が高い
- 海戦・長距離遠征に使う

---

## 操舵可能な3種類の戦闘艦

### Sloop（スループ） — 入門艦

建造コストが最も低く、最初の戦闘艦として選ばれやすい。

| 項目 | 内容 |
|------|------|
| サイズ | 超小型 |
| 建造コスト | Pinewood Plank×30 / Hemp Rope×10 / Iron Nail×5 / Canvas Cloth×2 |
| 特徴 | 最低コスト・軽快な操舵 |
| 向いている場面 | 最初の戦闘艦として |

---

### Ketch（ケッチ） — 小型帆船

Early Accessで主力として扱われる小型艦。

| 項目 | 内容 |
|------|------|
| サイズ | 小型 |
| 特徴 | 機動力重視、俊敏な旋回 |
| 火力 | 低〜中 |
| 耐久 | 低〜中 |
| 建造コスト | Wood×150 / Nails×100 / Coarse Fabric×40 / Rope Fiber×30 |
| 向いている場面 | 序盤〜中盤の探索・素早い移動 |

---

### Brig（ブリッグ） — 中型帆船

火力と機動力のバランスが取れた万能艦。

| 項目 | 内容 |
|------|------|
| サイズ | 中型 |
| 特徴 | バランス型 |
| 火力 | 中 |
| 耐久 | 中 |
| 建造コスト | 情報収集中 |
| 向いている場面 | 汎用的な探索・中盤の海戦 |

---

### Frigate（フリゲート） — 大型帆船

高い火力と耐久力を誇る主力艦。終盤の海戦で最強クラス。

| 項目 | 内容 |
|------|------|
| サイズ | 大型 |
| 特徴 | 火力・耐久重視 |
| 火力 | 高 |
| 耐久 | 高 |
| 建造コスト | 情報収集中 |
| 向いている場面 | 強敵との海戦・拠点防衛 |

アップグレード順の目安: **Sloop → Ketch → Brig → Frigate**

---

## Wharf（波止場） — 船の製造拠点

戦闘艦の建造・修理に必要な施設。

| 項目 | 内容 |
|------|------|
| 建造コスト | Wood×10 + Coarse Fabric×10 |
| 設置場所 | 水際（陸と海の境） |
| 機能 | 船の建造 / 修理 / カスタマイズ |
| カスタマイズ | 旗・帆・船体色 |

---

## 修理コスト

戦闘艦の **修理にはWood×20程度** が必要（艦種により変動）。遠征中に修理できるよう常備しておくことが推奨される。

---

## 船の召喚（スタータ艇）

- **Kキーでどこからでも呼び寄せ可能**
- スタータ艇は何度呼んでも無料
- 戦闘艦は Wharf のある港へ戻る必要あり

---

## 船カスタマイズ

各艦は Wharf で以下の項目をカスタマイズ可能:

- 旗のデザイン
- 帆のパターン
- 船体のペイント
- 装備砲の配置

**砲の種類**（現時点で確認されているもの）:
- 12-Pounder Cannon（Copper Ingot×10 + Wood×10）
