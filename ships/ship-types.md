# 船の種類

> 情報源: [Steam ストアページ](https://store.steampowered.com/app/3041230/Windrose/)

## 操舵可能な3種類の船

### ケッチ（Ketch）
小型帆船。Early Accessで入手できる最初の本格的な船です。

| 項目 | 内容 |
|------|------|
| サイズ | 小型 |
| 特徴 | 機動力重視 |
| 火力 | 低 |
| 耐久 | 低 |
| 向いている場面 | 序盤の探索・素早い移動 |

---

### ブリッグ（Brig）
中型帆船。火力と機動力のバランスが取れた船です。

| 項目 | 内容 |
|------|------|
| サイズ | 中型 |
| 特徴 | バランス型 |
| 火力 | 中 |
| 耐久 | 中 |
| 向いている場面 | 汎用的な探索・中盤の海戦 |

---

### フリゲート（Frigate）
大型帆船。高い火力と耐久力を誇る主力艦です。

| 項目 | 内容 |
|------|------|
| サイズ | 大型 |
| 特徴 | 火力・耐久重視 |
| 火力 | 高 |
| 耐久 | 高 |
| 向いている場面 | 強敵との海戦・拠点防衛 |

---

各船の詳細なステータス（帆速度・HP・砲門数など）、入手条件、アップグレードコストは情報収集中です。
