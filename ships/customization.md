# 船カスタマイズ

> 情報源: [Steam ストアページ](https://store.steampowered.com/app/3041230/Windrose/) / [GameRant 船カスタマイズガイド](https://gamerant.com/windrose-ship-customization-combat-guide-tutorial/)

## 外観カスタマイズ

Wharf（桟橋）の **Customization タブ** から外観を変更できる。

| 項目 | 内容 |
|------|------|
| **船名** | 任意の名前に変更可能 |
| **帆（Sails）** | デザインを変更可能 |
| **旗（Flag）** | デザインを変更可能 |

### 操作手順

1. Wharf に近づいてインタラクト
2. **「Customization」タブ**を選択
3. 変更したい項目（船名・帆・旗）を選んで設定する

## 装備アップグレード（性能カスタマイズ）

性能の強化は Wharf と Shipwright's Workshop の2か所で行う。

| 施設 | 役割 |
|------|------|
| **Shipwright's Workshop** | 装備品の製作・アップグレード（Bonfire と屋根が必要） |
| **Wharf** | 製作した装備を船に装着。水際に設置する必要あり |

> 船を管理する前に **K キーで船を Wharf に呼び戻す**（召喚）のを忘れずに。

### 主なアップグレード項目

| カテゴリ | 内容 |
|---------|------|
| **大砲（Cannons）** | 12-Pounder → 24-Pounder → 36-Pounder と強化 |
| **Hull Bracing（船殻補強）** | 耐久力強化。攻撃アップグレードより**先に優先**するのが推奨 |
| **Boarding Gear** | 乗り込み（Boarding）を可能にする必須装備 |

Hull Bracing のアップグレード素材は **Copper + Nails**（沿岸の難破船から回収）。

### 設計図の入手先

高性能な大砲・装備の設計図は Tortuga の派閥ベンダーから購入する。

| 派閥 | 販売品 |
|------|--------|
| **Rogue Buccaneers** | レア大砲設計図 |
| **Smugglers** | Hull Bracing 設計図・Naval Tactics 書 |
| **Brethren of the Coast** | 船（Brig / Frigate）設計図 |
