## ゲーム基本情報

- タイトル: Windrose（ウィンドローズ）
- 開発・販売: Windrose Crew / Pocketpair Publishing（日本）
- ジャンル: オープンワールドサバイバルクラフト + ソウルライク戦闘
- プラットフォーム: PC (Steam)
- Early Access開始: 2026年4月14日
- マルチプレイ: オンライン最大4人以上 / ホスト型セッション
