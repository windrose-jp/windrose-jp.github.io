# 用語集

> 情報源: [Steam ストアページ](https://store.steampowered.com/app/3041230/Windrose/) / [Steam コミュニティ ビギナーズガイド](https://steamcommunity.com/app/3041230/discussions/0/757304565299215807/)

本サイトで使用するゲーム用語のカタカナ・日本語対応表です。ゲーム内UIは日本語対応していますが、英語名も併記します。

## 戦闘・アクション

| 英語 | カタカナ | 説明 |
|------|----------|------|
| Stamina | スタミナ | 攻撃・回避・ダッシュに消費するゲージ。ゲームの根幹システム |
| Poise | ポイズ | 怯み耐性。削りきると敵（・自分）が転倒する |
| Parry | パリィ | 敵の攻撃を弾く防御アクション |
| Dodge | 回避 / ドッジ | 攻撃を回避するアクション |
| Chain Strike | チェインストライク | 連続攻撃システム |
| Bleed | ブリード / 出血 | スタック（最大5）で発動するDoT状態異常 |
| Critical | クリティカル | 会心の一撃。ダメージ増加 |
| Boarding | ボーディング / 乗り込み | 敵船に乗り移って行う船上戦闘 |
