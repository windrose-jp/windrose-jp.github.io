# 用語集（固有名詞対訳辞書）

本ページは **Windrose のゲーム内公式日本語訳と英語原文の対訳辞書**です。本サイトは海外コミュニティの英語情報を翻訳してまとめているため、**英語 → 日本語公式訳のズレ**が発生します。このページを参照することで、ゲーム画面で実際に見る表記と攻略情報を一致させられます。

## 表記規約

- 各ページの固有名詞は **「日本語公式訳（English）」** 形式で統一する（例: `千斬のレイピア（Rapier of a Thousand Cuts）`）
- **確度**列:
  - `◎` … ゲーム内で直接確認済み（スクショ・プレイ中）
  - `○` … 日本語攻略サイト・日本語配信で確認（複数ソース一致または信頼できる単一ソース）
  - `△` … 1箇所のみ確認、または意訳の可能性
  - `?` … 要検証（複数候補あり、または英語→日本語の飛躍が大きい）
  - `✕` … 未確認
- `?` / `△` 項目は英語原文を本文に残し、glossary で管理
- 公式訳が判明次第、glossary 更新 → 本文反映

## 情報源ポリシー

- 固有名詞の参照は**英語・日本語のどちらのソースからも可**（辞書的引用）
- 攻略テキスト・戦術・数値などの本文内容は**英語ソース翻訳のみ**（日本語サイト転載禁止）
- 主要参照: [GAMERS Wiki (jpngamerswiki.com)](https://jpngamerswiki.com/windrose/) / [Pocketpair Publishing 公式](https://www.pocketpair.jp/publishing/windrose/) / gamerch.com / konachangame.com

## 要検証リスト

以下はゲーム内で直接確認してほしい重要項目:

1. **Tumbaga → アルボルム**: 金属名が架空名詞に置換されている可能性大
2. **Tainted Priestess → 高位女司祭**: "Tainted"(汚染された) と「高位」で意味が異なる。別個体かも
3. **Greatsword → 両手剣 / グレートソード**: ソースで表記揺れあり
4. **Brigantine → ブリッグ船 / ブリガンティン**: 要確認
5. **Bonebreaker ↔ Crusher**: 片方が「クラッシャー」と訳されているが対応不明

プレイ中に実際の画面表記を確認したら報告ください → `◎` に昇格します。

## 更新方法

1. ゲームプレイ中に公式表記を確認 → このページに `◎` で登録
2. 日本語攻略サイトで同じ表記を複数確認 → `○` で登録
3. 本文ページを**一括置換**で更新（grep で英語原文を検索→日本語公式訳（English）形式に変換）
4. コミットメッセージは `[攻略] glossary更新: 〜の公式訳を反映` 形式

このページ自体は事実情報の辞書のため、著作権の転載問題は発生しない（個別の固有名詞は創作性が認められない）。
